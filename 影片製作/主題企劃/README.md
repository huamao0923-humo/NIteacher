# 主題企劃 — 資料夾說明

> 每支 20-30 分鐘 YouTube 影片的選題與敘事骨架。由 `/video-plan` 產出，依 CLAUDE.md 5.7 Schema。

## 檔名格式

`YYYY-MM-DD-主題.md`（日期為**企劃日**，非發布日）

例：`2026-05-01-取捨的智慧.md`

## 工作流程

1. `/video-plan 主題=XXX` 產出企劃頁
2. 使用者檢視、修改、確認
3. `/video-script 企劃=YYYY-MM-DD-主題` 進腳本階段
4. 企劃頁底部的「狀態」與「對應腳本」欄位由流程自動更新

## 不要手動新建檔

直接跑 `/video-plan`，避免跳過主題地圖 dedupe 檢查。
