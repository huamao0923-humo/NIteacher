# 腳本 — 資料夾說明

> 可拍攝的完整逐段腳本，20-30 分鐘長度。由 `/video-script` 四階段產出，依 CLAUDE.md 5.8 Schema。

## 檔名格式

`YYYY-MM-DD-主題.md`（沿用企劃日，方便配對）

## 四階段流程（仿 4.1 Ingest）

中間檔存 `_working/腳本-主題.md`，狀態列追蹤：

| Stage | 產出 | 停頓點 |
|-------|------|--------|
| 1 Outline | 章節大綱 + 時間戳 | 使用者確認結構 |
| 2 Draft | 逐段旁白，每段 `[x]` checkpoint | 中斷可續跑 |
| 3 Polish | 金句強化、字卡、鏡頭提示 | 使用者檢視節奏 |
| 4 Finalize | 移入本資料夾、更新進度表、補素材庫 | 完成 |

## 每段必含五欄

```markdown
### [時間戳] 段落名
- **旁白**：「...」
- **字卡**：「...」
- **鏡頭**：特寫/B-roll/桌面
- **情緒節奏**：...
- **金句**：「...」（縮圖候選）
```

## 與素材庫的關聯

Stage 4 Finalize 自動從腳本中蒸餾候選標題、金句、標籤，append 到 `素材庫/` 對應分片。
