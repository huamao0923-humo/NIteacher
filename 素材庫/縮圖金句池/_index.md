# 縮圖金句池 — 情緒索引

> YouTube 縮圖用的 3-7 字視覺強鉤子。按情緒分片。

## 分片規則
- 按情緒類型分檔：`震撼.md`、`好奇.md`、`反差.md`、`共鳴.md`
- 每檔上限 30 條，超出時按主題再分

## 情緒類別

- **震撼.md** — 衝擊、驚異、強翻轉
- **好奇.md** — 懸疑、疑問、未解
- **反差.md** — 反常識、反預期、對比
- **共鳴.md** — 痛點、情感、人生體悟

## 單筆格式

```markdown
- 「3-7字金句」— 來源 [[故事頁]]，適用主題：XX，狀態：未用／已用於 [[影片-...]]
```

## 使用方式

1. `/video-plan` 時，讀對應情緒分片當縮圖靈感
2. `/video-script` Stage 4 時，從故事頁金句中蒸餾 3-5 條追加到對應情緒檔
3. `/video-publish` 時標記已使用
