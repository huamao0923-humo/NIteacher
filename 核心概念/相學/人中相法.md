# 人中相法

## 定義與本質

**人中**：面部上唇正中央的溝槽，是鼻子下方到嘴唇之間的凹槽。

在相學中，人中的深淺、長短、寬窄，可以判斷：
- 生育能力（男女子嗣）
- 壽命長短
- 晚年運勢

---

## 倪師詮釋

### 人中深淺判斷生男女

| 人中形態 | 含義 |
|---------|------|
| 深（凹槽明顯） | 容易生男孩 |
| 淺（凹槽不明顯） | 容易生女孩 |
| 寬且長 | 子嗣多，生育能力強 |
| 窄且短 | 子嗣少，生育能力弱 |

### 人中與壽命
- 人中長而清晰 = 壽長之象
- 人中短或模糊 = 壽相較短
- 人中消失（因病或老化） = 壽元將盡之象

### 婆婆選媳婦看人中
- 傳統選媳婦的相法：看人中深淺判斷生男還是生女
- 深人中 = 生男兒，能傳宗接代 → 婆婆偏愛
- 此法來自倪師引用的傳統相術，現代可作為了解命格的參考

### 人中配合命盤
- 人中相須與紫微斗數命盤中的**子女宮**互相印證
- 若人中淺但子女宮星曜吉 = 生育仍有機會，只是比較辛苦
- 若人中深且子女宮吉 = 多子多孫之相
