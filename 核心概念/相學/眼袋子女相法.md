# 眼袋子女相法

## 定義與本質

眼袋（眼睛正下方的肉墊）是相學中觀察**子女運勢與父母對子女滿意程度**的重要部位。眼袋豐滿與否、左右是否對稱、有無特殊紋路，都透露出子女的狀況。

## 倪師詮釋

### 眼袋豐滿 = 子女令人滿意

> 「當一個父親以他兒子女兒為榮的時候，那個眼袋蓬起來很豐滿，他就對他兒子女兒很滿意。」

- 兩邊都豐滿 → 對兒女都滿意
- 左邊豐滿（兒子側）→ 對兒子滿意，對女兒不滿意
- 右邊豐滿（女兒側）→ 對女兒滿意，對兒子不滿意

（注意：左右側對應男女，需依實際相法慣例確認）

### 英子文（孩子過世的相）

英子文是眼袋下方出現的特殊「X」形或「×」紋：
- 在兒子眼袋側出現 → 兒子過世
- 在女兒眼袋側出現 → 女兒過世
- 紋路越深越濃 = 過世時間越久
- 剛過世時，眼袋下方呈**暗黃色**（非常黃）= 剛發生，傷心之相

> 「那你到華航去看做頭巡女，看看那個眉邊的顏色，你就給他上上香偷把一下。剛走的時候眼下袋可以看到……暗黃色很黃啊，很黃代表剛走啊，很傷心很傷痛啊。」

### 天羅文（配偶過世的相）

天羅文是出現在**額角**（髮際附近）的特殊紋路：
- 形如「X」形網狀紋，出現在天頂（額角）
- 表示太太（或配偶）已過世

> 「天羅，那個羅就是網嘛，那個蓋在天頂上，天頂上貼羅文，太太過世。」

## 面相識人補充（神氣識人法）

用人之道，看神氣是最可靠的方法：
- **眼神有威、無胸光** + 鼻樑很正 + 眼珠動都不動 → 可用，絕對不會背叛你
- **鼻子歪** + 眼睛閃動 → 準備被背叛
- **神很足的人**：坐在那裡動都不動，背挺直，氣場足

> 「你看那個人坐在那邊，他動都不動，背上提著鼻子的，動都不動了，這種人……從小開始，神很足。」
