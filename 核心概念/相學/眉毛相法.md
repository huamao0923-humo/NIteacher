# 眉毛相法

## 定義與本質

眉毛在相學中對應紫微斗數的**兄弟宮**，是觀察兄弟姐妹關係、人際合作緣分的外相指標。眉毛也反映一個人的性格、意志力與做事風格。

---

## 倪師詮釋

### 眉毛與兄弟宮

| 眉毛特徵 | 含義 |
|---------|------|
| 整齊清晰、眉色分明 | 兄弟和睦，互相扶持 |
| 雜亂、眉毛交錯 | 兄弟間易有糾紛口角 |
| 稀疏、眉毛少 | 兄弟緣淺，或是兄弟少 |
| 散亂、眉梢向下 | 與兄弟不合，或各自為政 |
| 眉毛高者（眉距眼遠） | **先走**（壽命較先配偶離世） |

### 眉毛高低與壽命
- **眉毛高者先走**（天紀第39堂再度確認）：眉毛位置高，離眼睛較遠的人，往往比伴侶先離世
- 眉毛低（緊貼眼睛）= 較晚走，多半長壽
- 批命時可用來判斷夫妻誰先離世

### 眉毛的形狀判斷

| 眉形 | 性格特徵 |
|------|---------|
| 劍眉（直而長）| 英氣勃勃，做事果斷 |
| 眉頭聚（皺眉紋深）| 思慮重，容易操心 |
| 眉毛彎曲柔和 | 溫和、重感情 |
| 眉毛粗濃（濃眉）| 心眼多，謹慎（對應天府星） |
| 眉頭扭曲（眉毛不順）| 與父母或兄弟有因緣糾葛 |

### 批命應用
1. 看兄弟宮星曜（紫微斗數）
2. 看眉毛形態（相學）
3. 兩者相互印證 → 確認批命準確
