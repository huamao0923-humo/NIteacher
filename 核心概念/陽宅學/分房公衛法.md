# 分房公衛法

## 定義與本質

分房公衛法是陰宅風水中判斷「祖墳格局利益哪一房子孫」的方法論。一塊墓地的龍虎砂水方位，決定了長房、次房、三房等各房的興衰吉凶，而非雨露均霑。

## 倪師詮釋

倪師指出：

**基本方位對應**
- 左邊（青龍）→ 長房
- 明堂正中 → 中房
- 右邊（白虎）→ 次房（三房）

**多子女分房口訣**（依砂案位置排序）：
- 一子：全盤滿印此一人
- 二子：左龍=長房，右虎=次房
- 三子：左龍=長房，明堂中=中房，右虎=三房
- 四子：清龍前峰上加入
- 五子：正案上另有一位
- 六子：白虎案上再延伸

**查「吃到風水」的方法**

子孫的長相要與墳地穴形相應，例如：
> 「王察察董事長，其祖墳站在猴形山前面，下一代出了個長相非常像猴子的後裔，才算真正吃到風水。」

**龍脈隻腳判斷發福代數**

龍脈前面伸出的「隻腳（支脈）」數量，代表發福的年數：
- 一條隻脈 = 管十年
- 十條隻脈 = 約100年（三代）
- 隻脈越多，福蔭越長遠

**凹風的警示**

若白虎側有缺口讓風進入（凹風），主該方敗絕。現場實測方式：站在不同位置感受風向，有凹風處即是凶位。

## 與其他概念的關係

- 與[[透地六十龍]]結合：透地龍的珠寶線要正，才能讓分房之福力完整發揮
- 與[[陽宅學]]對照：陰宅分房與陽宅卦位分房（長子居長子位）原理一致
- 與[[命象同參]]結合：需要「查資」（看後代長相）確認誰吃到風水，命相地三合一驗證

## 臨床應用

1. 看龍虎高低，判斷哪房氣勢較旺
2. 數龍脈伸出的支腳，預測發福代數
3. 查後代長相與穴形相似度，確認吃到風水的是哪一房
4. 注意凹風、洩氣、空亡位置，識別哪房敗絕
