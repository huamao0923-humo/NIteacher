# 名位相等

## 定義與本質
- **名**：你的身份、名義（例：公司負責人、一家之主）
- **位**：你實際居住或辦公的方位（例：西北角 = 乾卦 = 夫位）
- 核心公式：**名 = 位 → 加和（加分）；名 ≠ 位 → 扣分**
- 倪師金句：「你坐誰的位子要用誰的名字，位等於名才能發事業。」

---

## 倪師詮釋

### 核心邏輯
- 每個方位對應易經卦象，卦象有特定的「身份角色」
- 當你的名義（公司、家庭角色）與你實際所居的位置對應，事業與人生才師出有名
- 名不對位 = 出師無名 = 事業打折，甚至出問題

### 西北角（乾卦）= 夫位
- 丈夫住西北 = 乾為天 = 名位相等 = 事業旺、家和
- 若妻子住西北 = 地天泰（外順內剛），夫妻感情特別好，但事業主導易有偏差
- 若非夫妻主角住西北（如二兒子）= 名位不等，必有問題

## 實際應用案例

### 公司掛名應用
- 先生的名字 + 先生住西北角 = 事業運旺（名位相等）
- 先生的名字 + 先生住東南（長女位）= 天水宋卦 = 準備打官司
- 太太的名字 + 先生住太太位（西南）= 地火明夷 = 損女（女方受損）
- 解決方案：先生移回西北，公司改掛先生的名

### 節稅陷阱
- 夫妻為節稅，兩人各掛各的公司名字
- 結果：名不對位，必然一賠一賺
- 倪師臨床案例：6間公司4間掛老婆名、2間掛先生名，先生住西南角 → 老婆4間賺、先生2間賠，老婆還得補先生虧損

### 四代交替（傳承的名位之道）
1. **目前**：父母住西北（名位相等）
2. **傳給已婚兒子**：兒子夫妻搬到西北（名=夫妻，位=西北）；父母退到三兒子位（天山掛，享清閒）
3. **傳給未婚兒子**：兒子先住中宮，等結婚後再搬到西北
4. 若父母不退位、兒子繼承但名位不對 = 師出無名，事業必打折

## 看大宮發哪一房
- 一進門看哪個宮（房間）最大，就代表那一房（角色）最強、最發
- 東宮最大 = 長子最強；西南最大 = 婆婆說了算
- 倪師建議：先弄清楚哪間房最大，才能判斷家中主導者

---

## 女命應用

### 單身/離婚女性的位置選擇
- 離婚或單身女性不應住西南角（坤位 = 孤女之相）
- 想再婚：改住東南角（長女位 = 待嫁之象）
- 注意：改位時機要對，若逢化忌流年改位 = 二婚更凶

---

## 南北半球差異
- 北半球：西北=夫妻，東=長子，西南=母（此為基準）
- 南半球：南北方向對調，中間東西不動
- 先學好北半球，南半球一詞類推

---

## 辦公室應用
- 辦公室門方向（外卦）+ 老闆桌位置（內卦）= 組成一個易經卦象
- 門朝西北（乾）+ 老闆在西南（坤）= 天地皮卦（凶）
- 門朝西南（坤）+ 老闆在西北（乾）= 地天太（通暢，吉）
