# 赤鋁（長女補位效應）

## 定義與本質

**赤鋁**（音ㄔˋㄌㄩˇ）：易經八卦中，巽卦（☴）代表長女，對應東南方位。當家中未婚的長女長期住在**西北方（乾位/父親位）**，形成「長女入父位」的格局，稱為赤鋁現象。

易理依據：乾為天、為父，巽為風、為長女；長女住進父位，陰陽屬性錯位，影響家中的婚姻與命運。

---

## 倪師詮釋

### 核心效應：未婚長女住西北角

- 未婚的長女（或家中排行最長的女兒）住在西北角（乾位）
- 西北角是乾卦位，代表父/天/主/老闆
- 女性能量（巽/長女）佔據父位 → **補位效應**：她會承擔男主人的責任
- 結果：父親的能量被女兒替代，家中父親角色弱化

### 赤鋁的具體影響

| 影響面向 | 具體表現 |
|---------|---------|
| 婚姻 | 長女難嫁，婚事蹉跎；或嫁後仍控制娘家 |
| 父親 | 父親地位弱化，或健康走下坡，或過世較早 |
| 家中格局 | 長女像男主人一樣掌家 |
| 兄弟 | 若家中有兄弟，兄弟往往懦弱或出走 |

### 赤鋁的辨識方法
1. 未婚的女兒住在哪個房間？
2. 那個房間是否在西北方（乾位）？
3. 是否已是家中最長的女兒？
4. 三者符合 → 赤鋁成立

### 解法
- **最直接**：讓長女換房間，不住西北角
- 移到符合巽位的東南角，或符合她年命的延年方
- 若長女已婚出嫁 → 赤鋁自然解除（位置空出）

### 赤鋁與出嫁的關係
- 長女住西北越久，婚事越難成
- 因為她「補了父位」，從命格角度看，她已在家中「當主人」了，不容易嫁出去
- 解法：趕緊換房，或盡早讓她結婚出嫁
