# 透地六十龍

## 定義與本質

透地六十龍（簡稱「透地龍」）是陰宅風水中棺木入土時的精確方位系統。相對於「穿山六十龍」（用於格地面龍脈走向），透地龍專門決定**棺材在地下的擺放角度**。

共60個龍位，對應六十甲子，每兩個座山管五條龍，24座山×(60/24)≈5條。

## 倪師詮釋

### 三種龍的層次

| 名稱 | 使用時機 | 工具層 |
|------|----------|--------|
| 穿山龍 | 格地面龍脈走向 | 羅盤人盤（中針） |
| 透地六十龍 | 棺木入土方位 | 羅盤天盤（縫針）向內一格 |
| 格龍 | 看來龍格局 | 羅盤地盤（正針） |

### 六十龍的分類

- **珠寶線**（正線）：最吉，葬之兒孫發達
- **七三 / 三七線**：偏差，不佳
- **五線**：不好
- **火坑線**：兇，二十年後撿骨會見滿穴螞蟻
- **插手空亡線**：凶，「人才好善敗絕」

### 珠寶線的辨識

看羅盤天盤（縫針）向內一格：
- 顯示「正」字 → 珠寶線（最吉）
- 顯示「三七」 → 不吉
- 顯示「七三」 → 不吉
- 顯示「五」 → 不好

### 操作要點

1. 測定來龍方向（從水口看來龍，兩線交叉取焦點）
2. 在山頭移上移下，對準潮山，確定穴位
3. 棺木放入後，木碑與棺木需保持**九十度垂直**（方便日後驗測）
4. 確保棺木落在珠寶線上

### 犯凶線後果

倪師說：

> 「如果是火坑的線，葬之兒孫淫亂少亡，少年女嬌子，組孤寡，觀送不絕。如果大撞到插手空亡，人才好善敗絕。」

### 斷訟（判斷他人墳墓的透地龍）

若非自己測量的墳，可透過**木碑朝向**推斷透地龍位置，再對照吉凶批斷。

## 與其他概念的關係

- 與[[分房公衛法]]結合：透地龍正確才能使分房之福完整發揮
- 與[[陽宅學]]對照：陰宅透地龍=陽宅座向，同樣決定承接的氣場性質
- [[羅盤三層結構]]：透地龍使用天盤縫針

## 臨床應用

- 落穴前：用羅盤天盤向內一格，確認棺木落點是否在珠寶線
- 驗測他人祖墳：看木碑方向 → 對照透地龍 → 判斷兇吉 → 配合後代狀況驗證
- 合地移葬：若原穴在火坑或空亡，遷移時必須重新測定珠寶線
