# 天地匹（陽宅夫妻位互換格局）

## 定義與本質

**天地匹**（亦稱「天地否於家」）：在陽宅風水中，夫與妻的住房位置互換，造成諸事不順的格局。

「匹」取自易經天地否卦（皮卦）之概念，天在上、地在下本為自然之道，但若在陽宅中位置顛倒，即「匹」（不合、不通）。

---

## 倪師詮釋

### 基本定義
- **天地匹** = 夫在妻位、妻在夫位 → 陰陽顛倒，不合自然之道
- 和[[否卦]]（天地否）同義，應用在陽宅的夫妻睡房方位上

### 天地匹的四大凶應

| 凶應 | 說明 |
|------|------|
| 諸事不順 | 家中事業、財運、人事全面不順 |
| 身謙不成 | 謙虛退讓但沒有成果，努力白費 |
| 欺腎腹衰 | 腎臟、腹部相關疾病 |
| 得女 | 容易生女兒，難得男丁（取決於命格） |

### 判斷方式
- 夫的臥房是否在對應夫位（如西北乾位）
- 妻的臥房是否在對應妻位（如西南坤位）
- 若互換 → 天地匹格局成立

### 解法
- 調換夫妻睡房位置，回歸自然分配
- 配合[[延年方]]調整睡眠朝向

---

## 與其他概念的關係

- **[[否卦]]**：天地匹的易經原理來源，天地否的象意
- **[[陽宅學]]**：天地匹是陽宅方位學中的重要凶格
- **[[延年方]]**：解除天地匹需配合延年方位調整
