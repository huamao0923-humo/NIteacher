# 東宮生男法

## 定義與本質
- 東宮生男法是陽宅學的實用應用：利用**行房位置**（東宮東北/東南）來決定生男或生女
- 理論基礎：後天八卦的方位象徵——震卦在東=長子，巽卦在東南=長女

## 倪師詮釋

### 核心原理
後天八卦方位對應子女排行：
| 方位 | 八卦 | 象徵 |
|------|------|------|
| 東 | 震 | 長子 |
| 東南 | 巽 | 長女 |
| 南 | 離 | 次女 |
| 西南 | 坤 | 母 |
| 西 | 兌 | 三女 |
| 西北 | 乾 | 父 |
| 北 | 坎 | 次子 |
| 東北 | 艮 | 三子 |

### 生男的陽宅條件

**方法一：房間必須有東宮**
- 房子中心點的正東邊必須有臥室（不能是客廳或走道）
- 如果正東邊是客廳（缺東宮）→ 命中無子，換房子也難生男
- 東邊有房間但床靠東牆 → 壓著長子方位 → 生女兒

**方法二：床的位置**
- 床不靠東邊牆壁，靠北側或西側 → 生兒子
- 床靠東牆 → 生女兒
- 東南行房 → 生女；東往北行房 → 生男

**積極解法（命中無子者）**
- 找出家中東宮位置
- 在東宮（偏東偏北）行房 → 可補命中之缺
- 若客廳在東宮：暫時在客廳行房（倪師語：「客廳住是暫時的嘛，你只是一個月三天」）

### 先男後女 vs. 先女後男
- 東往北=男；東往南=女
- 可自行選擇想要的順序

### 二兒子補位法
- 二兒子住到長子的位置（東宮臥室）→ 二兒子行為思想變得像長子
- 若大兒子已結婚離家，二兒子自動補位成為「大兒子」

## 陽宅廚房方位的延伸
若廚房佔據特定方位，會「使（剋）」對應的家庭成員：
- 廚房在東宮 → 使長子
- 廚房在東南 → 使長女
- 廚房在坤方（西南）→ 使太太
- 廚房在乾方（西北）→ 使先生（夫妻死別）
