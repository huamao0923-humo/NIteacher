# 文昌文曲

## 定義與本質

文昌星與文曲星是紫微斗數的一組**讀書星（複星）**，倪師稱其為「複星」（成對出現才最有力）。兩星各有側重：文昌主科名（讀書考試），文曲主才藝（博學多能）。

---

## 倪師詮釋

### 文昌星的象意
- **主科學**：讀書考試、金榜題名
- 「牛年如果刚好今年逢到化科，或者是逢到文昌星，就是金榜提名。」
- 只有文昌沒有文曲 → **書呆子**
  - 學而不思，容易被人騙走
  - 「孔子說學而不思則罔（迷惘）」

### 文曲星的象意
- **主才藝**：博學多能，才藝豐富
- 「文曲星就比較博學，博學多能的人。顧名思義看起來就是比較斯文、文周周的。」
- 只有文曲沒有文昌 → 才多但考試不一定好

### 文昌文曲同時會命的格局
- **文采風流**（才子的象）
- 寫詩作文都很好，或才藝兼科名
- 女命同時會到 → 桃花名（批命時要委婉）
  - 「千萬不要跟她說小姐你有桃花，你要含蓄一點——男孩子都可以看到你很喜歡你啊。」

### 加上天魁天鉞（科甲星）
- 文昌 + 文曲 + 天魁 + 天鉞（四顆讀書星全齊）→ 「不管外表多粗鲁再壯，考試一定很厲害」
- 「千萬不要以為外形所獲。真的錯了。」

### 搭配流年的應用
- 文昌星入流年 + 化科 → 這年有考試中榜的可能
- 文昌 + 化忌 → 有考試機會但不中
- 「加減就出來了，學了以後看起來很複雜，其實很簡單。」
