# 四柱批命卦象總覽（天紀76-83堂）

## 定義與本質

本頁整理天紀第76-83堂所介紹的四柱批命法命卦名稱（卦象格局），這些「掛」並非易經標準64卦的卦名，而是倪師四柱推命系統中以「黃極經世」和「鐵板神速」排出的命盤格局，每個格局以特定名稱代表一種命理特質組合。

> 各掛的詳細批命內容（先天掛、後天掛、流年掛），請見各堂課程摘要頁。

## 特殊生肖影響規律

從各卦象的批命規律中，可以歸納出幾個反覆出現的生肖特性：

**雞年 / 屬雞**：
- 在多個掛象中都是「最吉」的生肖（鏡掛、水車結、三天大祖寶）
- 代表正面能量的承接

**鼠年 / 屬鼠**：
- 在頂掛中是大忌（遇鼠遭害）
- 在火折魁中鼠牛人平安

**虎年 / 屬虎**：
- 折火格逢虎貴人至
- 旭微風後天掛虎人難逃
- 水火未及流年掛虎人受困
