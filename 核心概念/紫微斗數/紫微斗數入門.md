# 紫微斗數入門

## 定義與本質

**紫微斗數**是中國傳統命理學中最重要的體系之一，以星辰入宮的方式推算人的命運。「斗」指星斗，「數」指其中的數理變化。

倪師在天紀課程中選用紫微斗數作為命理主線，原因是：
1. 可以把**天文**（命）和**地理**（陽宅/風水）融合在一起
2. 以年柱為主，比子平法更準確（子平法以日柱為主）
3. 可以化解（不像某些派別說命無法改）
4. 合理，可以讓各宗教和各界人士接受

---

## 倪師詮釋

### 易學難精
> 「紫微斗數易學難精。很容易進入狀況，很難精通。」

- 坊間版本太多，沒有一個中心
- 倪師系統從星性的**本質**（神）入手，不是記規則

### 命向同參
算命需要三個維度同時參考：
- **命**：紫微斗數排盤（天文）
- **向**：人的外相、當下氣色
- **同參**：加入地理位置，做加減

> 「跟你同時間生的人很多，住的地理位置不一樣，所以要命向同參。」

雙胞胎例：同年同月同日同時生，一個當醫師一個當廚師 → 地理位置（房間）不同所致

### 以果覺醒
> 「算命呢就是一個談果的哲學，它不談因。以果來決定它的行徑。」

- 先知道結果，再倒推如何行動
- 宗教都在談因，容易有偏差
- 算命是知道果，然後用地理位置、人事去化解

## 十二宮概覽

紫微斗數有十二個宮，代表人生不同面向：

| 宮位 | 代表面向 |
|------|---------|
| **命宮** | 本命、個性、外貌、整體命格 |
| **兄弟宮** | 兄弟姐妹關係 |
| **夫妻宮** | 婚姻、配偶特質 |
| **子女宮** | 子女數量與特質 |
| **財帛宮** | 財富來源與多寡 |
| **疾厄宮** | 健康、疾病 |
| **遷移宮** | 出外運、移居 |
| **僕役宮（交友宮）** | 朋友、部屬 |
| **官祿宮** | 事業、職業 |
| **田宅宮** | 不動產、家運 |
| **福德宮** | 一輩子的福祿，也看夫妻深層關係 |
| **父母宮** | 父母關係 |

## 主要星系

天紀課程涵蓋的主要星辰：
- **北斗主星**：紫微星、天機星、太陽星、武曲星、天同星、廉貞星
- **南斗主星**：天府星、太陰星、貪狼星、巨門星、天相星、天梁星、七殺星、破軍星
- **四化星**：化科、化權、化祿、化忌（見 [[四化星]]）
- **六煞星**：擎羊、陀羅、火星、鈴星、天空、地劫（見 [[六煞星]]）

## 星辰亮度：廟旺利陷

倪師以「妙望立現（陷）」代稱廟旺利陷：
- **廟（妙）**：最強狀態
- **旺（望）**：強壯狀態
- **利（立）**：平平狀態
- **陷（現）**：最弱狀態

煞星在陷 = 差中更差；煞星在廟旺 = 可能化煞為吉
