# 指揮星在引弓（兩大大貴格局）

## 定義與本質
- 指揮星在引弓是紫微斗數中兩種最高大貴格局的總稱
- 包含「武曲天府做環（直斧坐盤）」和「七殺朝斗格」
- 這兩種格局都代表掌握最高軍政權力的命格

## 倪師詮釋

### 直斧坐盤（武曲天府做環）

**格局條件**：
- 武曲（指揮星）在命弓（引弓）
- 天府星與武曲同宮
- 七殺在對面（三方會合）
- 四個角形成一個環形防禦陣→稱「環」

**象**：
- 四個警衛站在四角，守望防衛，每人可看到兩邊
- 這就是最有效率的防禦陣型

**結果**：**位列三台**
- 天子左右的最高參謀
- 古代叫「司馬、司空」
- 現代相當於宰相/將相之對
- 在天子旁邊輔佐決策

### 七殺朝斗格

**格局條件**：
- 七殺在命弓（丑宮）
- 向斗方（北斗/南斗）會合
- 整個格局像一個「斗」（古代盛米的容器）

**結果**：**將星入命，威震邊疆**

**批法的關鍵**：命弓的位置決定在哪裡威震
- 命弓在千宜宮（遷移宮）→ 到外地/邊疆威震邊疆
- 命弓在本地宮 → 待在天子旁邊，在本地成就

### 不同弓位的批法

| 命弓所在 | 批法 |
|---------|------|
| 命宮 | 最直接的大貴象 |
| 關路宮（官祿宮） | 在政府大好 |
| 財帛宮 | 到私人公司行號，當總經理/董事長更好 |
| 千宜宮（遷移宮） | 到外地發展更強 |

**批法原則**：「格式一樣，星座位置；宮不一樣，你批反高反的」→ 相同的格局在不同宮位有不同解法，要活批

## 與合八字的關係
- 這是第18堂的命盤舉例案例之一
- 此格局無論在哪個宮位出現，都代表極高的成就
- 若出現在官祿宮：在政府體制內大發
- 若出現在財帛宮：在私人企業更好
