# 外剛內柔命格

## 定義與本質

**外剛內柔命格**：紫微斗數命格中，命宮或財帛宮呈現「外表剛強、內心柔弱」的格局組合。

與易經兌卦「外柔內剛」（真悅）形成對比：外剛內柔是男命常見的特殊性格格局，表面強硬但感情細膩。

---

## 倪師詮釋

### 外剛內柔的命格組成

1. **命宮有殺曜**（如七殺、破軍、廉貞）= 外表剛強、意志力強
2. **同時有太陰或天同入三方**（或福德宮太陰強）= 內心細膩、感情豐富
3. 兩者並存 = 外剛內柔

### 外剛內柔的表現

| 面向 | 說明 |
|------|------|
| 外表 | 剛強、說話直接、不服輸 |
| 感情 | 內心細膩，重感情，捨不得離開 |
| 婚姻 | 感情上容易被人控制（因為太在乎） |
| 事業 | 外表強硬，實際上容易妥協 |

### 批命要點

- 外剛內柔命格的男性：事業上強硬，但感情上是「妻管嚴」
- 配偶往往知道他外剛內柔，反而主導婚姻
- 批命時需告知：「你外表強，但感情上要小心被拿捏」

### 外剛內柔與多婚的關係
- 外剛內柔 + 天喜在福德宮 = 感情上容易重複陷入
- 因為內心柔軟，捨不得傷人，多段婚姻中往往是被動的一方
