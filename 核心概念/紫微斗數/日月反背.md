# 日月反背（日月返輩）

## 定義與本質

### 廣義日月反背
- **基本格局**：太陽星落陷（夜生）＋ 太陰星落陷（日生）
- **象徵意義**：白天出月亮、夜晚出太陽，反常顛倒的格局
- **倪師比喻**：「命宮在這裡，白天出月亮——如果他是命宮的話，白天明明是太陽，他要指著太陽說這是月亮，你看看固不固執。」

### 父母宮日月返輩（特殊應用）
- **限定格局**：父母宮裡太陰化泉（化祿）＋ 太陽落陷
- **傳統論**：「惡性宜生」= 此命格適合過繼（與親生父母分離），才能活
- **現代論**：離開親生父母（生離或死別）
- **命格應驗**：命大者縱然不過繼，仍以其他方式應驗「與父母分離」（如被抱錯嬰兒）

---

## 倪師詮釋

### 廣義日月反背的三大影響

1. **工作節律顛倒**
   - 工作常白天閒、晚上比白天還忙
   - 適合夜班行業

2. **六親不靠**
   - 靠不到父母親
   - 各宮位表現不同（詳見下表）

3. **性情剛悍固執**
   - 白天指著太陽說是月亮 → 形容固執程度之深

### 日月反背在不同宮位的詮釋

| 宮位 | 具體表現 | 影響層面 |
|------|---------|---------|
| **命宮** | 此人固執、工作夜晚比白天忙、六親緣薄 | 性格與生活節律 |
| **父母宮** | 父母要離開（不在身邊或早亡） | 親子關係 |
| **子女宮** | 子女緣薄；男命子女遠離 | 親子下一代 |
| **夫妻宮** | 配偶不在身邊 | 婚姻關係 |
| **田宅宮** | 陽宅可能有火災（夜晚出太陽=大火） | 住宅與財產 |
| **官祿宮** | 做事業像晚上忙白天空，徒勞無功，不適合創業 | 事業發展 |
| **疾厄宮** | 眼睛有問題（青光眼或視力障礙） | 身體健康 |

### 父母宮日月返輩的現代案例

**天紀第30堂 — 被抱錯嬰兒**
- 民國47年命盤，父母宮有日月返輩之象
- 出生那天醫院發生火災，混亂中被抱錯嬰兒
- **結果**：他跟親生父母離散——以「被抱錯嬰兒」的方式應驗了日月返輩
- **倪師評語**：命格的事必然發生，只是方式不同。不需過繼，命大者自有其他方式應驗

### 相學驗證（日月角）

#### 正常人
- 日角（左額）、月角（右額）應稍微凸出

#### 日月返輩者
- 日角月角一平（既不凸也不凹）
- **倪師批命方法**：先觀察日月角面相確認後，才問「你有沒有跟父母住在一起」，避免傷害（此人心理敏感）

## 臨床案例

### 天紀第17堂 — 二十年連兇線
- 23～32歲大運：半空折煞
- 33～42歲大運：日月反背（太陰化泉）
- 連續二十年兇線（連兩個兇格）
- 加上殺度在夫妻宮 → 婚姻完蛋
- 父母離 → 婚姻衝 → 無子福
