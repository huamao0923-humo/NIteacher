# 四柱推命法

## 定義與本質

四柱推命（又稱八字算命）是根據人出生的年、月、日、時四個天干地支（共八字）推算命運的系統。倪師採用「黃極經世」和「鐵板神速」兩種算法，並以易經64卦圖作為批命核心，而非傳統的淵海子平法。

---

## 倪師詮釋

### 基本架構

四柱 = 年柱、月柱、日柱、時柱，每柱有天干、地支各一，共八字。

**倪師的核心差異**：
> 「排列的方法按照紫平法，批的方法完全是我自己重新批過——用易經的圖來批。」

各柱六親對應：
- **年柱**：父母
- **月柱**：兄弟姐妹
- **日柱**：夫妻（男查妻，女查夫）
- **時柱**：子女

### 天干取數（天干數）

| 天干 | 數 |
|------|----|
| 甲 | 6 |
| 乙 | 2 或 8 |
| 丙 | 8 |
| 丁 | 7 |
| 戊 | 5 |
| 己 | 1（或其他） |
| 庚 | 9 |

| 壬 | 2 |
| 癸 | 3 |

口訣：「甲六乙二丙八丁七，戊五己一庚九辛六，壬二癸三。」

### 求掛數（換算成易經卦象）

**天數（陽）**：
1. 取四柱天干，將單數（1,3,5,7,9）相加 = 陽數
2. 以 **25** 為基數，超過25就除去，餘數即天數掛號
3. 若恰好25，則取5

**地數（陰）**：
1. 取四柱地支，將雙數（2,4,6,8,10）相加 = 陰數
2. 以 **30** 為基數，超過30就除去，餘數即地數掛號
3. 若恰好30，則取0（重新計算）

### 先天掛（上卦下卦的決定）

**陽男陰女**：
- 天數（天干算出）→ 上卦
- 地數（地支算出）→ 下卦

**陰男陽女**（逆排）：
- 天數 → 下卦
- 地數 → 上卦

陽年生 = 陽人（甲、丙、戊、庚、壬年生）
陰年生 = 陰人（乙、丁、己、辛、癸年生）

## 鐵板神速算法

鐵板神速是以四柱推命批出6親命格的系統，共有 **12,000 條條片**。

> 「一萬兩千個條片——裡面的批法全部都是從64個卦圖裡面來的。有人說還有十幾萬條，那是自己附加上去的不需要。」

**計算流程**：
1. 算出先天掛
2. 確定要算哪一六親（父母/兄弟/夫妻/子女）
3. 用「乾宮加流度」（查父親）、「坤宮加流度」（查母親）等公式找到基礎數
4. 從數字對應12,000條片，找到對應的批命文字
5. 若初算不符，往前加5，或查對卦（對面之卦）核對

## 黃極經世算法

黃極經世（又稱「黃金經式」）以「原、會、運、勢」四層計算：

- **原會**：算父母
- **運勢**：算子女及六親

**計算特點**：
- 以天干和地支各自的數，採「順排」（原會）和「逆排」（運勢）兩種方式
- 順排代表地球順轉（從美國到台灣較快）
- 逆排代表地球逆轉（從台灣到美國較慢）
- 範圍比鐵板神速更大，更接近天地之數
