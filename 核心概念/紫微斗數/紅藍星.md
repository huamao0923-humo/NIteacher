# 紅藍星（天喜星 / 紅鸞星）

## 定義與本質

**紅藍星**：紫微斗數中的婚姻感情星，由紅鸞星（紅星）與天喜星（藍星）組成一對，合稱紅藍星。

- **紅鸞星**（紅星）：主動追求的一方，主感情主導，也主喜事到來
- **天喜星**（藍星）：被動接受的一方，主因他人而喜，帶來外在的喜慶機緣

---

## 倪師詮釋

### 核心功能
- 紅藍星入夫妻宮 = 該大限逢婚姻機緣，是**婚年訊號**
- 同時看哪個星先入、哪個星後入，判斷婚姻主導權

### 天意星的涵義
- 倪師有時稱紅藍星為「天意星」
- 含義：婚姻是天意安排，由天意星決定何時結婚
- 天意星入夫妻宮 = 天意要你結婚

### 女招貴婦 / 男娶美妻
- 紅藍星配合文昌文曲（才藝星）+ 太陰 = **女招貴婦**格
  - 此格局的女性：氣質出眾、招來有社會地位的婆婆/女性贵人
- 紅藍星在福德宮（對應夫妻宮）= 此生婚姻由天意定，強求無效
- 男命有此格：配偶容貌出眾（娶美妻）

### 判斷婚年
1. 看大限：哪個大限紅藍星同時入夫妻宮？
2. 看流年：在那個大限內，哪一年流年再沖動紅藍星？
3. 兩者疊加的那年 = 婚年

### 紅藍星在各宮的意涵
- **夫妻宮**：直接婚姻機緣
- **福德宮**：情感深厚、緣分是命中注定
- **命宮**：感情影響整個人生格局
- **子女宮**：感情中容易有子女先來（未婚先孕可能）

## 與其他概念的關係

- **[[四化星]]**：化祿、化科入夫妻宮配合紅藍星 = 婚姻更順遂
- **[[天喜星]]**：紅藍星中的藍星，詳見天喜星獨立頁面
- **[[合八字]]**：婚姻除紅藍星外，還需配合對方命盤合八字
- **[[夫妻宮]]**：紅藍星的主要作用宮位
