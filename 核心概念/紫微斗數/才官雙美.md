# 才官雙美（科全祿命格）

## 定義與本質

**才官雙美**：紫微斗數命格中，才能（文昌/文曲）與官位（科名/官祿）同時具備，形成文才與官運兼備的美好格局。

又稱「科全祿命格」，代表一個人既有才能，又有實現才能的官位資源。

---

## 倪師詮釋

### 才官雙美的組成要素

1. **文才**：文昌或文曲在命宮或官祿宮入廟/廟旺
2. **官位**：化科、化祿、官祿宮有吉星護持
3. 兩者同時俱備 = 才官雙美

### 才官雙美格局的特徵

| 方面 | 表現 |
|------|------|
| 事業 | 靠才能升遷，在專業領域有成就 |
| 社會地位 | 有官職或高職位 |
| 聲名 | 以才名聞，而非純靠權力 |
| 財運 | 財來自才能回報，非不勞而獲 |

### 科全祿命格的進一步說明
- **化科**（科名星）入官祿宮或命宮 = 才華被社會認可
- **化祿**（財星）同時入 = 才能帶來財祿
- 科+祿雙入命/官 = 「科全祿」= 才官雙美的最高級別

### 文昌文曲 + 太陰 = 女命才藝格
- 文昌文曲代表文才、藝術
- 太陰代表陰柔、美麗、感受力
- 三者同現 = 女命才藝格：才藝出眾（音樂、舞蹈、繪畫等），氣質優雅
- 女性有此格：不靠容貌靠才藝，以才藝立身

---

## 才官雙美 vs 一般文才

| 格局 | 有才 | 有官位 | 結果 |
|------|------|-------|------|
| 只有文昌文曲 | ✅ | ❌ | 懷才不遇 |
| 只有化科 | 部分 | ✅ | 官位但才藝平平 |
| **才官雙美** | ✅ | ✅ | 才展其志，名實雙收 |
