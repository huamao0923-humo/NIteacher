# 女命太陽性反背

## 定義與本質

**女命太陽性反背**：女性命盤中，太陽星在命宮或夫妻宮且呈現「反背」之象（落陷或化忌），導致女性性格出現「陽性化」傾向，或對父夫子的關係出現異常。

這與[[日月反背]]（太陽夜晚出現/太陰白天出現）有關，但專指女命太陽的特殊反背現象。

---

## 倪師詮釋

### 太陽對女命的意涵

- **太陽**主：父親、丈夫、兒子（三父夫子之象）
- 女命太陽吉 = 與父/夫/子關係良好，靠父夫子得福
- 女命太陽凶（落陷/化忌）= 與父夫子緣薄或不靠

### 女命太陽性反背的表現

| 表現 | 說明 |
|------|------|
| 性格陽剛化 | 女性卻有男性氣質，決斷、主導、不服管 |
| 夫緣弱 | 嫁不到好丈夫，或丈夫早離（生離或死別） |
| 獨立不靠男性 | 事業自立，不依賴父夫 |
| 父緣薄 | 與父親關係疏遠，或早失父 |

### 判斷方法

1. 女命命宮：太陽落陷（子宮/丑宮）= 太陽無力，性反背象
2. 女命夫妻宮：太陽化忌 = 丈夫有化忌之象，婚姻有損
3. 結合面相：日角（左額）低平或不凸 = 父夫子關係異常（見[[日月反背]]）

### 解法

- 陽宅調整：讓女性住在能接收陽性能量的方位（乾位、東方）
- 讓女性腳朝太陽有力的方向（太陽廟旺的宮位對應方位）
