# 天喜星（多婚婚姻象）

## 定義與本質

**天喜星**：紫微斗數中的感情喜慶星，是紅藍星（紅鸞+天喜）的組成之一。天喜星代表喜慶、婚緣、生育喜事的到來。

天喜星在不同宮位，有不同的感情與婚姻應用。

---

## 倪師詮釋

### 天喜星的基本星性

- 天喜星入夫妻宮 = 該大限有婚姻機緣（與紅鸞同入 = 婚年）
- 天喜星代表「喜慶事件」到來：婚事、生子、喜宴

### 天喜星在福德宮 = 多婚象

**最重要的特殊應用**：

- **天喜星落在福德宮** = 多婚命格
- 福德宮代表一生的深層情緣和福分
- 天喜多入福德 = 多次婚姻的緣分；情感上不斷有新的「喜慶」
- 表現：婚姻次數多（兩次以上），或感情複雜多段

#### 進一步細化

| 天喜星位置 | 含義 |
|-----------|------|
| 夫妻宮 | 婚年（當運出現時結婚） |
| 福德宮 | 多婚命格，一生感情喜慶多 |
| 命宮 | 感情豐富，但未必多婚 |
| 子女宮 | 子女喜事多，或先有孩子後結婚 |

### 多婚男命案例（第40堂）

- 男命天喜星在福德宮 + 夫妻宮有連破星（或地結星）
- 批命結果：此人必有多次婚姻
- 案例應驗：第一段婚姻因某種原因結束，第二段又起
- 關鍵：不是道德問題，而是命格本身的緣分模式

### 紅藍星與天喜星的區別

| 星 | 代表 |
|----|------|
| 紅鸞星 | 主動追求的感情力量，婚前情感波動 |
| **天喜星** | 婚後喜慶，多次婚姻機緣 |
| 合稱紅藍（紅+天喜） | 婚年指標，共同構成婚姻大限判斷 |
