# 半空折尺

## 定義與本質
- 某宮的主星入廟（亮），但對面宮位有化忌來沖，且三方四正無助力
- 象徵「竹子長到一半折斷」= 中途夭折的命格
- 定律：**凶運必在中年**（命盤中間那段大限，約30～45歲）

## 倪師詮釋

### 核心公式
```
某宮主星入廟（明亮）
+ 對面宮位化忌沖來
+ 三方四正無助力星
= 半空折尺
```

### 為何凶在中年
- 命盤中間那段大限 = 中年期
- 凶災不需細看，先鎖定「線」在中年
- 即使命格其他部分很好，此格局必應在中年一段時間

### 先天 vs 後天
- 先天：本命盤有半空折尺 = 一生中年必逢凶
- 後天：大運走到某宮，形成半空折尺的臨時格局

## 各堂講述內容

### 天紀第21堂的定義
- 某宮的星入廟（亮），但對面宮位有化忌來沖
- 加上三方四正無助力時，稱為「半空折尺」
- 凶運必在中年（30歲左右），先天或後天都可能發生

### 天紀第17堂案例（日月反背搭配半空折尺）
- 23到32歲：半空折尺
- 33到42歲：日月反背（畫記太陰）
- 連續二十年兇線，婚姻崩潰，無子福，六親離

### 天紀第26堂——流年化忌與半空折尺
- 流年走到命宮對面化忌來沖 = 最凶
- 太陽星現（落陷）= 眼睛看不到日月 = 昏迷或死亡之兆

### 天紀第39堂——半空擇次（40歲前的判斷界限）

- **半空擇次**：半空折尺的凶線有一個重要判斷點：**40歲前**是最危險的時期
- 若凶線（半空折尺的大限）在40歲前過完 = 已渡過最危險期，後半生相對平順
- 若凶線延伸到40歲後 = 仍有風險，需持續關注
- 「擇次」= 選擇（凶線）的下一段，判斷後半生的格局

批命要點：
1. 先確認半空折尺的中年大限在哪個時間段
2. 判斷那段大限是否在40歲前結束
3. 若已結束 = 告知命主最危險時期已過
4. 若尚未結束 = 繼續配合陽宅調整

### 天紀第30堂——民國42年案例
- 命盤：半空折尺 + 火星 + 千宜（畫記）
- 畫記 = 半空折尺（非常凶）
- 凶災鎖定中年
- 民國82年，中南部某政治人物被槍擊，41歲（中年大限）
- 流年：太陽現地（落陷），日月看不到天上的光 = 死亡應驗
