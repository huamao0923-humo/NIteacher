# 深宮（深功）

## 定義與本質
- 深宮（又稱深功、身宮）是紫微斗數中代表**後天最大影響宮位**的概念
- 與命宮（先天性格）不同，深宮反映你的人生實際走向與後天環境的影響

## 倪師詮釋

### 命宮 vs. 深宮
| | 命宮 | 深宮 |
|--|------|------|
| 代表 | 先天性格、本質 | 後天最大影響 |
| 特性 | 先天注定 | 後天環境塑造 |
| 比喻 | 你是什麼人 | 什麼因素影響你最多 |

### 深宮在各宮的意義

| 深宮所在 | 後天影響意義 |
|---------|------------|
| 官祿宮 | 後天在官場/政府機關發展；仕途是最大影響 |
| 財帛宮 | 後天在私人企業做事，工作賺錢是人生重心 |
| 遷移宮 | 後天必須到外地發展，離鄉是宿命 |
| 夫妻宮 | 後天受婚姻影響最大；配偶影響命運走向 |
| 福德宮 | 後天受個人信仰、修養影響最大 |
| 田宅宮 | 後天受家庭、房產影響最大 |

### 命宮與深宮同宮（生命同宮）
- 先天（命）與後天（深宮）一致，命運簡單明確
- 例：命宮在財帛，深宮也在財帛 → 天生就是賺錢的命，後天也走財路

### 深宮逢七殺（最忌）
- 「七殺、鈴星，中部美」
- 深宮最忌逢七殺——後天環境逆境重重

## 臨床應用
- 批命時，先看命宮定先天性格，再看深宮定後天走向
- 深宮所在 = 建議這個人應該往哪個方向發展（官場/商場/外地/婚姻）
- 案例：命中有男命象（如七殺+天梁）的女性，深宮在財帛 → 必須做職業婦女，靠自己支撐
