# 本地外地論（出生地與發展地）

## 定義與本質

**本地外地論**：在紫微斗數批命中，判斷一個人是否應該在出生地（本地）發展，還是離開本地（去外地）才能發達。

這與命盤中的**遷移宮**星曜和**官祿宮**的特性有關。

---

## 倪師詮釋

### 判斷標準

#### 適合在外地發展（離鄉得發）
- **遷移宮有吉星**（如化祿、天馬、紫微、天同）= 出外有貴人，宜離鄉
- **天馬在命宮或遷移宮** = 天生愛動，不適合定居本地
- **出生地與陽宅磁場不合** = 離開本地後命運改善

#### 適合留在本地（本地發達）
- **田宅宮強** = 在本地有土地、房產緣，宜留守
- **遷移宮有化忌或煞星** = 出外有損，不宜離鄉
- **官祿宮與本地磁場吻合** = 在本地事業有成

### 發字（兒童命盤）的特殊說明
- 「發字」：倪師用來稱呼小孩出生後第一個命盤的特殊術語
- 小孩命盤尚未走到大限，先看本地/外地的基礎格局
- 出生命盤有天馬動象 = 這個孩子長大後宜外出求發展
- 出生命盤官祿宮吉 = 先看本地，再評估外地

### 本地外地與陽宅的配合
- 即使命盤適合外地，陽宅磁場也要配合
- 若本地住宅方位佳，可先改善陽宅再決定去留
- 三才（天命+地理+人事）三者配合，決定最終走向
