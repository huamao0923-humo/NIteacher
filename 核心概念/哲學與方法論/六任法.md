# 六任法（引弓法）

## 定義與本質

六任法（又稱引弓法）是倪師傳授的一種**掌上即時吉凶判斷法**，源自諸葛亮行軍出兵時的占卜方法。只需知道當天農曆月、日、時，即可快速判斷「此行/此事吉凶」。

> 「以前諸葛亮帶兵打仗，出兵的時候看他到底是不是好日子。」

## 使用方法

### 所需資料
- 農曆月份
- 農曆日期
- 當前時辰（子丑寅卯…）

### 六任名稱（依序）
```
大安 → 留蓮 → 速喜 → 赤口 → 小吉 → 空亡
```
（黃曆上有記載，但一般人不知道如何正確使用）

### 計算方式

以農曆月份為起點，對應六任名稱排列，再以農曆日和時辰推算到對應的任。

### 六任含義

| 任名 | 吉凶 | 含義 |
|------|------|------|
| 大安 | 吉 | 約一週可見結果，大體平安 |
| 留蓮 | 凶 | 停滯不前 |
| 速喜 | 吉 | 約兩週見結果，喜事快來 |
| 赤口 | 凶 | 有第三者介入，小人是非，人為因素 |
| 小吉 | 吉 | 小利可期 |
| 空亡 | 大凶 | 此事不可為，去也無用 |

## 使用時機與準則

倪師強調，此法適合在**心血來潮**時使用：

> 「當你心血來潮的時候，想用它的時候就要以當時為準。」

適用場景：
- 突然想做某件事（買股票、簽合約）
- 出行前有疑惑（要不要搭這班車/飛機）
- 交出去一件事的吉凶（談生意、升遷命令）
- 找失物、測薪水來潮之事

**關鍵守則**：
1. **問了就要聽**：問了結果又不照做，書上說「涉水滅頂」
2. **不能同一件事問兩次**：算完後若換時辰再問，反而混亂
3. **空亡則不動**：抽到空亡，直接放棄此事

## 與氣色結合使用

倪師提到，在機場或公眾場所，可結合**氣色相法**一起判斷：
- 看到周圍人臉色多黑，加上六任落空亡 → 高度警覺，考慮不搭此班次

## 倪師金句

> 「問了就要聽他。問了又不聽，易經上怎麼說：涉水滅頂。你自己去涉水然後跑得很深，沒人害你，自己被自己害死的。」
