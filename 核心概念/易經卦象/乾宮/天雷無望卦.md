# 天雷無望卦

## 定義與本質

天雷無望卦（☰☳）為上乾下震，天在上、雷在下。「無望」並非「沒有希望」，而是「製成」（誠實面對）之意。

> 「無望本身這兩個字的意思是製成。製成的意思。所以當君子知道復的時候，我們要以製成的心、製成的態度才能夠通行無阻。」

## 倪師詮釋

### 無望 = 製成 = 自成

「自成」是天雷無望卦的核心精神：

> 「你開車把人家撞死了，沒有關係，不要跑，通通要以自成的態度來面對他，任何難關都可以過。」

- **自成**：承認錯誤、以誠信面對，即是「動之以天（正理）」
- 小人「動以人欲」= 因自己的慾望而動 → 招凶、自取其害
- 君子「動以天（正理、時機）」= 內剛健、外堅實 → 能大積

### 無望卦的象理

- **內卦震（動）**：要動，以正理而動
- **外卦乾（天）**：動必以天、以正理
- 外象非常堅實、內個性剛健 → 能夠大積（大蓄）

### 聽聲辨人：從丹田還是喉嚨

天雷無望卦給出識人方法：
- **從丹田發出的聲音**：實話、真誠、自成之人
- **從喉嚨發出的聲音**：造虛偽、造假，心有不實

> 「你用易經64個卦來看人，就變成64個像。真正自成的人，他到你面前，他講話的內容很實在，然後很面對你。」

## 占卜圖像

天雷無望卦的圖像：一個鹿在上下、老鼠、豬、怪人射箭、水中兩堆錢。

| 圖像 | 含義 |
|------|------|
| 官人射鹿（文官） | 自亂陣腳 |
| 鹿 | 財（祿鹿）；財訓到了 |
| 錢在水中 | 得財遇險 |
| 鼠+豬（黑色） | 有陰謀的合夥人 |

## 陽宅應用：三子居父位

天雷無望卦的陽宅應用稱「三千大樹」：
- 三兒子（或三女兒）住到爸爸（西北乾位）的位置
- 凶應：帶父出征、才激大飲（很小氣）、婚姻短暫（不足三個月）、做生意平平、當官閒官
