# 否卦（天地否卦）

## 定義與本質
- 否卦 = 易經天地否卦（第12卦），代表「閉塞不通」
- 外卦：天（乾，往上走）；內卦：地（坤，往下走）
- 天在上繼續往上，地在下繼續往下 = **兩相隔絕，不通**
- 易經本文：「匹茲匪仁」= 不是仁者之道，不是正道

## 倪師詮釋

### 卦象的哲學
- 否卦 = 外表剛強（乾），內心散漫（坤往下）
- 泰卦（地天泰）= 外順內剛，與皮卦完全相對
- 一念之間：外柔內盡 = 太；外似君子內壞水 = 皮
- 「人一念，天地知」——內心狀態決定你是太還是皮

### 泰卦到否卦：物極必反
- 泰卦到達終極（上六），陰在外繼續揉柔 = 反轉成皮卦
- 太平盛世之後，一定要防範反轉的到來
- 君王若一昧柔順（只有揉，無威）= 最後小人得道
- 「物極必反，太卦到極點就是皮卦」（倪師語）

## 皮卦時代的特徵

### 小人當道，君子道消
- 皮卦 = 小人當道的時代
- 在疲惑（皮賽）時：「所有小人都是升官的」
- 外面穿西裝筆挺、說的都是君子話，裡面一肚子壞水

### 識別小人的方法
- **無視而笑**：見到你不說話，就在那裡笑 = 心生毒氣，是壞蛋
- 言行不一致，表裡不一致
- 看到你總是拍掌幸災樂禍 = 造假造謠的第三者

## 皮卦占卜圖像解讀

### 圖像一：男子臥病（羊病）
- 圖中男子臥床 = 裝病
- 含義：一事無成，以裝病應對；或政治人物「裝病退休」= 閒才之道

### 圖像二：進破（鏡子打破）
- 鏡子打破落地 = 破鏡難圓
- **夫妻難和**，再難團圓
- 若有招（徵招入朝）= 不可復（不要去接受），否則被利用完後丟棄

### 圖像三：小人當道（人坐路中）
- 一群人坐在路中間，不讓人通過
- 你已被利用完了，趕快退出
- 張弓箭頭在地 = 最後一劍已用

### 圖像四：拍掌笑（第三者造謠）
- 旁邊幸災樂禍地拍掌笑 = 第三者造謠破壞
- 識別方法：鄰居中哪個人見到你不說話就笑？= 就是他

## 除皮之道（君子的應對策略）

### 基本原則：以退為進
- 遇到皮賽不通 = 君子**以退為進，不居榮位**
- 繼續留下拼搏的都是小人
- 能退出來的，才是有智慧的閒才

### 履卦解皮
- 處皮之時，有禮（履）就可解：「對人謙和有禮」
- 天則履（履卦）的序卦是地天太（太卦）
- 逆向推：處皮之時用禮，可引回太的狀態

### 在政壇的應用
- 遇到皮賽不通 = 留下的都是小人（升官）
- 有識之士應趁早退出，「處皮之時歸閒才」

## 陽宅應用

### 辦公室風水
- 辦公室門朝西北（乾）+ 老闆在西南（坤）= 天地皮卦（凶）
- 辦公室門朝西南（坤）+ 老闆在西北（乾）= 地天太（吉）
- 老闆桌的方向決定辦公室成皮還是成太
