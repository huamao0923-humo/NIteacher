# 太掛

## 定義
太掛是易經的第一一卦，代表陰陽交合、天地通暢、和諧融通的狀態。

## 倪師詮釋
太掛代表的是天氣下降、地氣上升，陰陽在相交的狀態下通暢流動。

## 卦象的自然現象
- **陽在上**：天氣下降
- **陰在下**：地氣上升
- **交合**：陰陽相交，天地通暢

### 日常例子
- 太陽照射地面 → 地面水氣蒸發
- 地氣上升為雲 → 形成雲的循環

## 外順內界的應用
- **外順**：表現謙和有禮、戰戰兢兢
- **內界**：內心有實力、每天至強不息
- **結合**：謙和有禮 + 內在充實 = 太掛

## 倪師的自我評價
「我一路上就是這樣——機天太掛」
- 內心充足、實力非常
- 外面謙和有禮
- 這種人最終會得到「太」的結果

## 包荒的君王智慧
- **包荒**：容納所有粗糙、醜陋的事物和人
- **氣量**：有足夠的胸懷接納各種人事物
- **歷史例子**：經國先生到麻風病院握手
