# 地火明夷

## 基本資訊

- **卦序**：第 36 卦
- **卦象**：坤（上） / 離（下）
- **核心意象**：光明傷於地中、韜光養晦

---

## 定義與本質

地火明夷（☷☲），上坤下離。第36卦，火地晉之後。「明夷」即光明被傷。

- 火在地下 = 光明被埋沒、被遮掩
- **反卦**：火地晉（火在地上）反過來 = 地火明夷（火被地埋沒）
- **象象含義**：火原在地上（晉卦日出地上），現在火跑到地下去（光明被埋）

---

## 倪師詮釋

### 象象的雙重含義

**第一層象象**（自然之象）：
- 火在地下 = 光被埋沒、昏暗之卦
- 與火地晉形成對比：晉卦是日出地上（光明），明夷是日出地下（昏暗）
- 火性往上、地性往下 = 反向運動 →演變為「豬猴之象」（火跑到地下發不出光）

**第二層象象**（人事之象）：
- **暗君之卦**：君官在上位、做事暗地里搞陰謀
- **明者見傷**：光明之人反而受傷
- 具體現象：「玩陰的。玩完陰的以後每天拿著聖經去唱和，『我是基督教的，我是聖經的門徒，我怎麼會說謊話』——最壞的（虛偽）」

### 序卦邏輯

火地晉（盛） → 地火明夷（必傷）
- 盛到極致，物必衰；衰退之象就會傷
- 「物盡不一樣，必傷。一定會受到傷害。所以我們常常不是說一步都很順隨。物盡了以後一定會有傷到」
- 比喻：汽車引擎日久會磨損，這是自然之理

---

## 象象的分析

### 核心原理

**明夷之「傷」的本質**：
- 傷於外：外人伤害（小孩打架回家找媽媽）
- 傷於己：明者見傷（光明之人因為太明而遭暗算）

**君子與小人的區別**：
- **君子在明夷之時**：
  - 外順（配合、順應） = 避禍災
  - 內明（心知肚明） = 自治、自我要求高
  - 體現「和而不同」精神：內有原則、外有應變
  
- **小人之象**：
  - 玩陰謀、表面敬虔內心虛偽
  - 最難識別、最具迷惑性

### 君子「悔而明」之戒

**關鍵概念**：明察秋毫反而生禍，需要「悔而明」不是「全明」

**經典比喻**：
- 「水太清澈就沒有魚在裡面」
- 「人自清是無土的，水自清是無魚的」
- 過度明察導致人心遠離，終成孤立

**具體應用**：
- 不要把什麼都查出來、什麼都去搞
- 「事過成半，都不足輕重，算了，好像微罪不理了」
- 某些細微過錯可「得過且過」，無須窮根究底
- 達到「警告」「處罰」的目的就夠了，不需要無所不察

**為什麼**：
- 無法明察天下所有事件（ABC三戶都有做錯，查A清楚、查C清楚、B給他放掉 → B反而懷疑你偏心）
- 過度明察反生禍：「失重之道」（人心不齊、信任瓦解）
- 結果變成：「C就會說比較矯心一點」（被反咬一口）

### 六爻人間道：詹甫（六爻象象）

根據占卜圖像解讀：

1. **婦人在井中** = 代表女人遭災 / 家內有災（井脖子之象）
2. **老虎在外面看** = 外人虎視眈眈、競爭威脅
3. **錢象變兩倍** = 財商往來、財路相關
4. **人族路** = 一個人在趕路、有私財之相
5. **針香未露** = 針代表財路、尚未露頭
6. **補到** = 如果占卜結果是此象，代表逐路終圓、揭竿而起的局面

## 臨床應用

### 陽宅對應

**關鍵應用**：女性住在火位（南邊 = 離位 = 地火明夷）

**具體凶應**：
1. **健康**：隱疾難言（不好意思講的病、婦科病等）、破財
2. **性格**：異獸小人言（易信謠言、妒忌心重）
   - 現象：「聽人家的三姑六婆講話」→ 妒忌夫妻太好 → 製造謠言
   - 結果：太太信以為真，先生一回來就先修理一頓 → 眼尾就掛了（夫妻離異之象）
3. **諸事不順**
4. **見血光避凶**：開刀都不好、擴散、狀況惡化（不是開刀本身不好，而是開完刀狀況變差）
5. **婚姻極凶**：先生把舊妻趕走（不要錢），新妻卻陷在井中（置陷進去、改不了）

**夫妻位置凶應案例**：
- 先生住西北（乾位 = 天），妻子住南邊（離位 = 地火明夷）
- 隔壁鄰居妻住地水師 + 自家妻住地火明夷 → 兩家都要出事
- 現象：「夫妻兩個先生身體很好，太太身體很差」或反之

**位置調整解法**：
- 若發現妻住南邊（地火明夷）導致家凶
- 可讓先生改住南邊（天火同人 = 好格局）
- 妻子改住別的位置（坤位或中間）
- 「位置可以隨心調整的，懂不懂」
- 有人住西北不爽、大兇，後來跑去住西南被劈出來（更兇） → 需要聰明調整

### 人事判斷

**識人應用**：
- 暗君在上、玩陰謀之象
- 表面敬虔、虛偽最深
- 不可親近 / 必須「和而不同」以保護自己

**決策應用**：
- 遇到明夷時機，不要太銳意進取
- 外順內明 = 等待、蓄養實力
- 不可一意孤行、逼迫改革
