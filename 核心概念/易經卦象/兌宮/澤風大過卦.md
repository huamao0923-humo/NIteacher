# 澤風大過卦

## 定義與本質

澤風大過（☱☴）為上兌下巽，澤在上、風（木）在下。「大過」有兩層含義：

1. **過之大**：犯了大的過失
2. **大過於人**：超越常人所能做到的成就（非凡之舉）

倪師用「折風大過」這個說法（沼折木），象徵沼澤（澤）淹沒樹木（巽/木），樹木被水滅的象。

## 倪師詮釋

### 兩種大過的辨別

- **小人的大過**：犯錯後放不下，「是非越多，越來越僵」——過之大的惡性循環
- **君子的大過**：「舉世非之，全天下都在罵他，他不怕；舉世不知文，而無悔。這種人都能夠立不失之功勳。」

歷史典範：毛澤東等大人物，都有「全天下罵你也不理」的氣度（大過於人）。

> 「你有人在罵你，你跑去花三個小時跟他解釋，腦筋不要放在人的上面。」

### 頤→大過的連結

> 「養成以後必動，動則有過。所以大過呢，就是經歷過養這個階段進來的。就好像小孩子，養成以後他會動，動的時候有時候太過了，杯子打破啊，小到杯子打破，大到外面就闖禍。」

這是頤卦（養）到大過的自然過渡——養之後必動，動則有所過失（成長的必要過程）。

### 占卜應用（折風大過圖像）

| 圖像 | 含義 |
|------|------|
| 官人乘車上插兩旗 | 官司訴訟；或武官軍人 |
| 旗上喜字（兩個分開） | 婚姻破損，可以捨去 |
| 入豬門 | 豪門世家；或姓豬（姓家）|
| 門外貴人立 | 棄、去以外之象；談判沒空間，可以捨掉 |
| 文書在地 | 合約無法約束；合約不成 |
| 盒子在正南上方 | 先成後破之象 |

### 陽宅應用

- 折風大過卦（長女住西鄉）= 戀愛轟轟烈烈沒有結果
- 女孩子若住長女位（東南宮）= 成格，但個性任性剛強，嫁大她十歲的人
- 三牛兒住在大牛兒位置（夫妻年紀接近）= 婚姻有兇
- 男子住進去 = 早婚三年，嫁夫年長十歲以上，深入豪門

### 大過卦識人（外象）

> 「你發現你太太突然對你很喜悅而且百依百順，你又做錯了大事怎麼回事？你兒子搞不好玻璃瓶打破了，或者你太太可能錢被他偷走了，不敢跟先生講，突然變得很喜悅。奇怪今天是什麼日子？一定是折風大過。」

外順內虛 = 有事隱瞞的外象。
