# 火風鼎

## 基本資訊

- **卦序**：第 50 卦
- **卦象**：離（上） / 巽（下）
- **核心意象**：格物致知、讓不相融的相融、耳聰目明之道

---

## 定義與本質

鼎卦接在澤火革（改革）之後，是「革而後鼎」——改革之後需要以「鼎」的智慧整合融合。

「鼎」是烹飪器具，能使火與水（本不相融之物）透過中間媒介（食材）得以相融。引申為「讓對立的事物得以調和」的能力。

---

## 倪師詮釋

### 鼎的真意

> 「中國人過去把那個鼎，食物米放在裡面，那個鼎的腳在下面撐著，下面是火，上面是水。水和火本來是不融的，可是這個鼎在中間能夠讓它相融——所以鼎卦的意思就是如何讓不融的能夠相融。」

**比喻**：法官辦案，兩造對立，法官就是「鼎」——讓兩邊不融的立場，透過公正中立得以調解。

### 鼎人的條件

「耳聰慕名（目明）」是鼎人（能做調和者）的首要條件：
> 「A來跟你說他有道理，B來說也有道理——那兩個回去還是找你，你沒有用。鼎一定要耳聰目明，知道人家哪裡犯的錯誤，讓對方曉得錯在哪裡，氣就消了，和解了。」

### 鼎象的三特質

1. **忠實（中間填充）**：內部是很有才的人，裝了很多東西
2. **行為端正如鼎**：行為端正，四平八穩，不偏側
3. **至善至道**：終身不會有過，是正道

### 敗亡之道（鼎卦的警示）

> 「德博而未尊（職務高沒有功勞），知曉而謀大，利少而任重——這個就是君王用非其人，遭敗亡之道。」

### 公正如鼎

> 「背斯為公——把你私人的慾望、私人的想法放到背後，用人為公。用的人要對社會有很大貢獻，就像軍方將領，一定要他有很多功勳，讓手下幸福。」

## 補卦（占卜）象

當卜到火風鼎時：

| 象 | 意義 |
|----|------|
| 雲中月線（月從雲中出） | 事情顯現，好事漸明 |
| 波雲劍月 | 清零、時節（清明/冬至） |
| 卻南飛 | 冬至來臨，北人執政（論國運） |
| 一紙裹席帽 | 無知之人介入，或人才不得其用 |
| 直刀站中間 | 五官護衛將軍，執行之人 |
| 貴人無畏 | 軍王、法官、無私老闆 |
| 老鼠在下 | 陰謀、暗算、屬子年者 |

---

## 與革卦的連卦關係

澤火革（改革） → 火風鼎（建立新秩序）：
革卦打破舊有格局；鼎卦在改革後建立新的融合機制，是「格而後立」。
