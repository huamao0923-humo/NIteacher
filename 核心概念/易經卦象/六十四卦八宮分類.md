# 六十四卦 — 八宮分類

> 依**上卦（外卦）**分類，64卦分為八大宮，每宮 8 卦，共 64 卦。
> 排序方式：每宮以「純卦（上下同卦）」為首，其餘依卦序號排列。

---

## 分類邏輯

```
每一卦 = 上卦（外卦）× 下卦（內卦）
上卦 = 8 種 → 8 個宮
每宮 8 卦 = 上卦相同 × 下卦各不同
```
