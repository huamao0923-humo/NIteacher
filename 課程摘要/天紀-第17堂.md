# 天紀 — 第17堂

## 日期／來源
天紀課程 第17堂

## 核心主題
**八字批命中的實務技巧——出生時辰判斷法與紫微丑宮格局分析**

## 關鍵概念列表

### 出生時辰不清楚時的判斷法
- 許多人不知道準確的出生時辰
- 可通過「法序位置」來推斷：
  - **植物牟友時生**（早上5-7點）：法序居中或中央偏左
  - **隱身事害時生**（下午3-5點）：法序偏右
  - **程序走位時生**（晚上7-9點）：法序有雙迅或多迅

### 十二地支與方位的對應
- 某時＝正東（3-5點）
- 午時＝正南（11點-1點）
- 酉時＝正西（5-7點）
- 子時＝正北（11點-1點）

### 方位應用
- 八字與陽宅（風水）結合時，需要知道12個宮位（對應12個方向）
- 羅盤有24個分精，但實際應用只用12個
- 其他用途與地理（山川河流）相關

### 紫微星丑宮格局
- 紫微星在丑宮時，科全入廟的星曜跟隨
- 破軍一定在旁邊（紫微破軍相伴）
- 其他星曜（陀螺、天府、太陰等）配置相同

### 命宮空亡的特殊格局
- **半空則次**：命宮為空，對面是連胎
- 這是「天羅地網」格局
- 中年線（30歲左右）會有夭折現象
- 可能是先天帶病到該年發病，或環境造成自殺

### 天相星的含義
- 天相＝天上的宰相，是輔佐人才
- 天相心在命宮：位高無權，善於輔佐
- 女孩子得天相心最好
- 天相人一定是瘦高型、高挑型

### 宮位與十年大運的應用
- 十年大運時看當年的宮位
- 45-54歲大運在天相宮位：該十年內會「位高無權」
- 例如做副院長、副主委等輔佐角色

### 左腹右臂的含義
- 紫微星若有左腹右臂：極其貴重
- 左腹右臂在千宜宮時：外地比本地更吉利
- 反之則本地比外地更吉利
- 批命要反推理，靈活應用

### 夫妻宮中的林侦探
- 林侦探＋天罗地網＝婚姻極差
- 兩個桃花星落陷在夫妻宮＝必然有九色才細
- 此格局下婚姻必有大問題

### 杀破狼遇林侦探——各種死法
七殺、破軍、探郎是殺星，再遇到林侦探（天羅地網），殺加殺：
| 組合 | 結果 |
|------|------|
| 林侦＋七少（林侦西上） | 半路死亡 |
| 林侦＋破君 | 水中作坟（水裡溺死/水中謀殺） |
| 林侦＋探郎 | 腰折（先天病/自殺） |

- 若杀星**入廟**（很亮）：眼看見殺，是看得見的危險（如鄰居出事）
- 若杀星**落陷**：是無預警的凶事

### 武曲七殺（無曲其策）的多重含義
武曲七殺同宮，長的是「拳」不長「鹿（財）」：
| 情境 | 結果 |
|------|------|
| 女命 | 孤單——孤軍奮戰 |
| 男命 | 離家到異鄉發跡 |
| 武職（軍警法外交）男 | 武職大力 |
| 商人 | 勞多功少 |

### 陽宅位置影響夫妻緣分
- **西北客廳**：先生位置是客廳=先生像客人（生離）→ 做船員/飛行員/軍人之命
- **西北廚房**：廚房是刀的象=夫妻死別（此在第14堂已詳述）
- **東邊客廳**：兒子變客人（不孝順/愛出門）；或命中無子

### 子女宮化忌的陽宅解法
- 命中無子 + 東邊是客廳 → 百分之百夭折
- 解法：把東邊改成臥室（東宮）→ 可補命中之缺
- 天人地三才各佔1/3：陽宅佔了重要角色

## 🎬 二創素材

### 故事1：七殺破軍探郎遇林侦探——命中各種死法
- **震撼點**：倪師把七殺/破軍/探郎分別配上林侦探，說出三種不同的死法——半路死亡、水中作坟、腰折自殺
- **應用**：命盤看到杀星組合的實際意義；杀星入廟vs落陷的區別
- **倪師金句**：「杀星很亮就代表你眼睛看得到的杀——你家隔壁生活了，你看到了」

### 故事2：西北客廳=先生的旅館
- **角度**：你家西北方是客廳，先生就像住旅館一樣——永遠在外面
- **倪師金句**：「所以我常常在算命的時候，我家住這種房子，我說你家能夠開個牌子，叫做某某市大旅館。先生就像客人一樣。」
- **反常識**：不是先生不回家，是房子的格局造成的

### 二創應用建議
- **獨立短影片**：「七殺破軍探郎——三顆殺星遇上林侦探，各有不同的死法」（建議3-4分鐘）
- **延伸影片**：「你家西北方是什麼？客廳=先生不回家，廚房=先生去世」
