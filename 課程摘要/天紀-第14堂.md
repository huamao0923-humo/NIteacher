# 天紀 — 第14堂

## 日期／來源
天紀課程 第14堂

## 核心主題
**命宮與各宮位的具體分析——以太陽星、相學面相為案例**

## 關鍵概念列表

### 十年大運的宮位應用
- 十年大運推進時，看當年落在哪個宮位
- 該宮位代表當年最主要的人生事件
- 例：34-43歲的十年落在田宅宮，則需看田宅相關的吉凶

### 太陽星的含義
- 太陽星代表先生（父親）或陽性象徵
- 太陽在命宮：父親福德
- 太陽在子女宮：兒子
- 太陽星落陷在子女宮且過了某個年份會「看不到」，意味著陷入困境或分離

### 十年大運定位的原則
- 34-43歲直隸在某宮，該宮就是當年最影響的人生面向
- 流年推進與十年大運重疊時需特別注意
- 40歲直隸在夫妻宮時，該年夫妻會有大憲

### 相學面相細節
- **眼角尾部（魚尾紋）**：代表婚姻狀態
  - 一條兩條：夫妻恩愛，滿意度高
  - 三條以上：不推薦
  - 打十字紋：夫妻死別之象
  - 往下走的紋路：婚姻不好

- **眉毛立起**：丈夫剛回家時眉毛立起 → 剛做完壞事

### 五官命的特殊現象
- 五官強旺的女孩子（如天相命）容易得到「男命女生」的評價
- 男命女生：女性卻有男性氣質和決策能力，容易感到疲勞

### 凶宅——陽宅兩大殺局
1. **廁所在房子中間**（不靠任何牆邊）
   - 象意：心臟悶住的象——無窗、無氣通
   - 後果：住戶易患心臟病、中風（動風）、氣喘
   - 原理：廁所靠邊有窗可通氣，居中則氣悶——取像看精神，不看形

2. **西北角是廚房**
   - 西北角 = 乾位 = 父（先生）的位置
   - 廚房 = 刀的象（切割、刀鬥）
   - 象意：夫妻死別之宅
   - 同理：廚房在坤方（西南）→ 克妻；廚房在東宮 → 克長子；東南 → 克長女

### 東宮生男法（陽宅決定生男女）
- **東邊 = 長子的方位**（後天八卦：震在東）
- 原理：行房位置決定生男女（依九宮方位）
  - 床靠東邊牆壁 → 生女（因為壓著長子方位）
  - 床移離東牆，靠北側 → 生男
  - 東南行房 → 生女；東往北行房 → 生男
- 積極解法：命中無子的女性，在東宮處行房（暫時），可補命中缺陷
- 客廳在東邊者（缺東宮）：可暫時在客廳行房，補位

### 相學面相補充
- **人中寬窄**：人中寬廣 → 好生養；人中細如線 → 易難產；人中平坦無溝 → 無法生子
- **魚尾紋（眼角）**：最佳一到兩條；三條以上不建議；十字交叉紋 → 夫妻死別；紋路往下 → 婚姻差
- **眉毛立起**：剛回家時眉毛豎起 → 剛做了不可告人的事（可作驗命工具）
