## 關鍵概念列表

### 一、日月格局詳解

#### 1. 日月夾財（吉利型夾命）
- 命宮在醜宮或未宮，太陽太陰同時各自入廟，夾住命宮
- 特徵：財路無缺，一生榮華
- 命宮內若有吳曲（武曲）貪狼等五官星 = 夜半出五官星
  - 女性：女中豪傑（如敢衝到抗議前線的人）
  - 男性：白天的五官星才是正科正將

#### 2. 日照雷門 / 月天天門 / 明珠出海格
- **日照雷門**：太陽在卯宮（早上5-7點），東方正掛，正為雷，太陽剛升起最亮
- **月天天門**：太陰在亥宮（晚上9-11點），月亮正當頭
- **明珠出海格**：命宮在醜宮或未宮，三方恰好同時會到日照雷門（太陽）和月天天門（太陰）
  - 最高格局，命有此格自帶日月雙明
  - 一詞類推：非命宮也可，落哪個宮即批那個宮

#### 3. 日月併明格
- 太陽太陰同時在命宮的對宮或三方四正會照
- 主一生榮華，事業如日不落國
- 日月併明的人24小時通明，睡覺做夢都能看清是非
- 明珠出海格也屬日月併明的一種

#### 4. 太陽在害宮或子宮（犯水桃花）
- 太陽在亥（害）宮或子宮 = 「貪居害子」
- 犯水桃花，桃花命
- 女命得此：名氣很大，大家都喜歡
- 男命得此：妙望時出武將，落陷時能阻殺

#### 5. 陰心入廟（破軍在武宮）
- 破軍在午宮（武宮）= 陰心入廟
- 與破軍在子宮同義，相對宮位
- 男命：威震邊疆，非常陰挺
- 女命：瘦小型，孤僻，對感情不重視，婚姻較晚

#### 6. 子撫作環 / 七殺朝斗
- 紫微在申宮，七殺在寅宮（相對）= 子撫作環或七殺朝斗
- 一輩子絕路農場（社會地位和財路都有）

### 二、易經地水師卦占卜詳解

#### 地水師卦占卜中的關鍵圖像
- **羊回頭**：若卜病人屬羊，得地水師 = 病人會從陰間回陽間（康復）
- **圍棋盤**：代表事情越做越多（可成），象棋盤代表越來越少
- **文武兩官授印**：有文武兩官支持
- **藏人級**：要得到大家信服的人才來指揮
- **將軍台台立**：手有兵權，主生殺大權

#### 地水師陽宅應用
- 媽媽住正北方 = 地水師（每天出門凶悍）
- 太太住正北方先生在同房 = 媽媽是師、爸爸是宋（天水訟），夫妻衝突

### 三、水地比卦（親和之道）

#### 自然現象的啟示
- 水與地永遠連在一起，是天地間最親和的組合
- 老祖宗從此悟出「比之道」= 親和之道

#### 人間道的基礎原則
- **同欲相求**：欲望相同的人必然走在一起
- **同憂相親**：憂慮相同的人自然親近

#### 戰國策外交智慧（故事延伸至第23堂）
- 中山國張登的計謀：利用各國的「同憂」讓齊王孤立
- 最好的外交官是看透人性的人

## 🎬 二創素材

### 故事1：病人的屬相決定卦意 — 羊回頭
- **類型**：易經占卜實用 / 命理神準
- **核心**：同一個「地水師卦」，若問的人屬羊，代表「羊回頭」= 從陰間回陽間 = 病人康復
- **戲劇性**：一個卦，因問者屬相不同，結果天壤之別
- **二創建議**：「易經占卜為什麼要問屬相？」——解析地水師卦圖像的活用法

### 故事2：明珠出海格 — 日月雙明自帶光環
- **類型**：紫微斗數格局 / 命理辨識
- **核心**：命宮在醜宮或未宮，同時會到日照雷門（太陽在卯）和月天天門（太陰在亥）= 生來自帶日月雙光，睡覺做夢都通明
- **金句**：「日月並明的人24小時通明，睡覺做夢都能看清你是小人。」
- **反常識**：這不是後天修練，是命盤裡自帶的「識人之眼」
- **二創建議**：「為什麼有些人天生就識人？」——從命理解析日月並明格

### 故事3：太陽在害宮犯水桃花
- **類型**：紫微斗數格局 / 桃花命理
- **核心**：「貪居害子」——太陽在亥（害）宮，犯水桃花。女命名氣大，人緣極佳；男命能阻殺
- **二創建議**：「桃花命真的是命？」——用太陽落宮解析人緣格局的形成原因

### 故事4：斧頭疑心病——坎位讓人行險
- **類型**：哲學寓言 / 陽宅坎位效應
- **核心**：樵夫斧頭不見，懷疑鄰居偷竊，看誰都像賊；找到後立刻覺得鄰居正常了——住坎位就是這樣的心態
- **金句**：「人住在坎位就是行險，什麼事情都覺得有問題。」
- **頁面**：[[斧頭疑心病——坎位讓人行險]]

### 故事5：父子觀棋——看神不看形
- **類型**：哲學智慧 / 觀察力與洞察力
- **核心翻轉**：父親看棋子（形），兒子看趨勢（神）——圍棋越下越多，象棋越下越少。孩子的天真問題比大人更有智慧
- **金句**：「我們要看神不看形。」
- **頁面**：[[父子觀棋——看神不看形]]

### 故事6：兒子住坎位偷竊——陽宅決定行為模式
- **類型**：陽宅案例 / 坎位效應
- **核心**：富家子弟不缺錢卻愛偷竊（享受行險感）；倪師進屋看到床在坎位（正北方），直接斷言此人必有法律問題
- **金句**：「你們家誰睡這張床，誰就會一直有官司是非，法律問題纏身。」
