# 天紀 — 第19堂

## 日期／來源
天紀課程 第19堂

## 核心主題
**易經卦象的人生智慧——從蒙卦、水天需卦看成長與修養**

## 關鍵概念列表

### 山水蒙占卜圖解——滿船珍寶先乘後破
- **圖中元素**：兩人乘船，船上滿是珍寶；岸上有財（陸地）；中間放了個墳墓；旁邊有牆；上有月亮
- **解讀**：一僧人引路入空門（古代跑路避官司）；月當天=因人居位（女皇帝/女主掌權）
- **墳墓**：墳墓裡有棺材，棺＝官，材＝財，**棺材=財富**
- **結果**：若用蒙蔽手段，「先乘後破」——整個計畫終究失敗
- **人間道**：為政者不可有貪念，要做大事不做大官，知行合一

### 人生成長的卦象序列
- 乾為天、坤為天 → 兩個結婚
- 懷孕 → 水雷屯（妊娠期）
- 出生 → 山水蒙（啟蒙期）
- 成長 → 水天需（需要養成）

### 水天需卦的含義
- 需＝等待、養成的時間
- 物質成長時必須「外現內界」（外面簡陋，內心豐富）
- 這是養成自己的正確方式

### 外現內界的實踐
- 把房子弄簡陋破爛，自己專心讀書
- 小偷都不會來（外現破爛）
- 能專心致志，不受幹擾（內界充實）

### 古人的修養典範
- 古人住在茅草屋裡讀書 → 能靜心受教
- 躲在山洞裡讀書 → 小偷找不到，內心安定
- 這是「外現內界」的智慧應用

### 水天需卦的天人合一觀
- 水往上升 → 雲蒸潤現象
- 對應人體：喝水 → 身體吸收 → 嘴巴出汗
- 正常現象：喝水後有汗液分泌
- 異常現象：喝完嘴巴還乾 → 可能是糖尿病或心臟問題

### 天人合一的應用
- 天地現象和人體現象是相通的
- 易經理論可應用到中醫、生活方方面面
- 中醫的理論根據就是這種天人合一

### 剛見之人的修養
- 「剛」＝有原則，有自己的主張
- 「見」＝知識豐富、有見識
- 剛見之人最容易犯錯：以為自己能力強，急於求成
- 越是剛見之人，越要懂得「待時而進」

### 待時而進的智慧
- 不要急躁、求功、冒進
- 要等待合適的時機
- 這才是「至善之道」

### 對庸才的警示
- 庸才常常說「時不與我」，怨天尤人
- 頻繁換工作（一年十二個月換十三個工作）
- 以為自己多有能力，實際上不懂得「適應環境」
- 應該改變自己去適應環境，而不是等環境來適應你

### 台灣民心不古的根源
- 人們失去了「待時而進」的耐心
- 人們失去了「外現內界」的修養
- 都在期待「適合自己的工作」，而不是「適應工作」

### 天水訟卦——訴訟的源頭與脫險之道

**卦象結構**：外天（見）、內水（顯）→ 外見內顯

**訴訟的三種命格**：
| 格局 | 結果 |
|------|------|
| 賤而不顯（從不行險） | 不申訟（一輩子不會打官司） |
| 賢而不見 | 不能訟（根本無法達觀世） |
| 賢而又見 | 訟也（真正的官司糾紛） |

**訴訟的根源**：外見內顯（外表好看/裡面空洞）
- 借五十萬開名車來→外見內顯→送之源
- 一天到晚做人際關係，內心空洞→也是外見內顯

**脫險之道**：「淨險」——面對問題，而非逃避
- 險事發生，要立刻出面處理
- 逃避問題只會使問題擴大
- 「人入險地而不知」就會乘訟

**柳性——人屈人伸（君子之道）**
- 柳樹風再大，隨風搖晃，就是不斷
- 韓信胯下之辱=柳性=大丈夫，能屈能伸
- 如此才能避訟（能夠忍一時之氣，避免無謂糾紛）

### 陽宅：正北方=坎=行險格局

- **正北方（坎卦）**：坎是水，是行險之道
- 住到正北方的人：容易官司是非（天水訟）
- 父居二子位（正北）→ 必訟（延遲五年的禍）：
  - 32歲本該化權，因住在坎位延到37歲才掌權
  - 延時五年是坎位代價

**月未現象（二兒子住爸爸位置）**：
- 智慧老成得太快（行為思想像40歲的人）
- 婚事延後，或娶二婚妻/年齡差大的伴侶
- 或根本不想結婚（以為自己已是爸爸的角色）

## 🎬 二創素材

### 故事1：墳墓=棺材=財——顛覆對死亡的理解
- **反常識**：大家都怕墳墓，但在易經占卜圖裡，墳墓代表財富（棺=官，材=財）
- **倪師詮釋**：「墳墓裡面一定有棺材，棺就是棺，材就是材，棺材！」
- **應用**：風水/命理看到墳墓象不一定是壞事

### 故事2：韓信胯下之辱——柳性的智慧
- **震撼點**：易經天水訟卦說「柳性」——韓信被逼胯下之辱，沒有還手，這就是君子之道
- **倪師金句**：「人屈人伸，大丈夫！韓信啊，人受胯下之辱。」
- **應用**：車子碰撞先道歉，不管誰對誰錯，就是避送之道

### 故事3：通緝犯跑到廟旁警察局——脫險之道實例
- **幽默點**：倪師說如果是通緝犯，要跑到最靠近警察局的廟旁邊每天吵鬧，讓警察以為他只是煩人的平民
- **倪師金句**：「你跑到山裡面去，他派人出來坐在路口看誰跑到山裡面去；結果這邊沒有派人——脫險之道！」
