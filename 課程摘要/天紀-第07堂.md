## 關鍵概念列表

### 1. 四化星概覽

四化星是紫微斗數的「主力星」：
| 化星 | 意義簡述 |
|------|---------|
| **化科** | 科名、名氣、才藝 |
| **化權（全）** | 官印、信剛、權力 |
| **化祿（錄）** | 財祿、福德 |
| **化忌（記）** | 忌諱、阻礙、耗損 |

**重要：四化星是固定的，不動的。**

> 「我們只要排出方式，固定在那個宮上面，不要動。流年在動，星不動。」

坊間有「飛星」派（每年科權祿忌在不同位置飛來飛去），倪師認為此法錯誤，越弄越複雜。

### 2. 化科（科）詳解

**主要代表：**
1. **科名**：考試金榜題名、官位科第
2. **文章才藝**：文筆好、畫畫好、才藝超群
3. **教學**：當老師也是科
4. **名氣大**：生命遠播、聲名遠播

**看手相判斷有無化科：**
- **天柱紋**：手掌上一條紋路從下往上一路通天
- 天柱紋的人主科、主才藝（如畢卡索等藝術家必有此紋）
- ⚠️ 「博士眼」（手指上的眼形紋）不是真正的科，只是形象

**名氣的時間點：**
- 科在命宮 → 一生有名
- 科在某大運 → 那段期間名氣大
- 科出現很晚 → 可能死後才出名（如某些畫家）

### 3. 化權（全）詳解

**主要代表：**
1. **官印**（觀印）：官職、蓋章的權力
2. **信剛**：個性剛強、信念堅定
3. 在不同宮位有不同的應用（需搭配宮位判斷）

例：夫妻宮有化權（信剛）→ 婚姻中一方強勢，不肯讓步

### 4. 四化的宮位應用原則
- 每顆化星落在不同的宮，代表那個宮位的面向被強化
- 要判斷「是哪顆星化科（或化權…）」→ 看那顆星再決定發生什麼事

### 5. 化祿詳解

**主要象意：**
1. 財祿（財富的來源，信守財）
2. 先天本性若是化祿入命 → 守財奴（一毛不拔，把錢藏起來）
3. 看化祿在哪個宮：父母宮有祿 → 父母從商；財帛宮有祿 → 自己做生意；命宮有祿 → 主業自然到身上

**看手相：**
- 男：鼻頭（財帛宮）——絲筆/懸膽筆者主財
- 女：手伸出來枯枯乾乾燥燥的 → 大財婦相

### 6. 化忌詳解

**主要象意：**
1. **劫殺**（阻礙、橫禍、官司糾紛、破財）
2. **想不開**（個性面：悲觀、鑽牛角尖）

化忌的人**外貌特徵**（面相）：
- 五官擠在臉中間（眉毛、眼睛、鼻子全皺在一起）
- 眉毛太低、眼睛太靠近鼻子 → 一輩子想不開

> 「看那大官、財路好的人，相都是眉毛打開、五官寬大的。五官都皺在一起的，一輩子想不開。」

人間道應用：化忌的念不對 → 外界小人攻擊 → 輪不到該有的機會

### 7. 氣色判斷（相學：望氣）

**喜事氣色（吉）：**
- 黃潤色（像黃玉一樣潤潤的）→ 主喜事、喜慶來臨
- 賺大錢時眼睛瞇著，眼角旁黃潤

**化忌氣色（凶）：**
- 非常黑暗、灰暗，好像沒有洗臉
- 灰塵感，有「毀孕」之象

**死色（臨死徵兆）：**
- **青藍色** 或 **枯萎色**（頭髮乾蓬無油光，皮膚不反光）
- 看到此色 → 兩個節氣（約14天）內有危險
- 倪師親見案例：太太看到先生臉色發青，打完高爾夫球後當場倒地身亡

> 「看死色可以討論很多事。你只要有戒心，再大的災都可以去掉。」

### 8. 北斗星系入門

**紫微星（北斗星君，第一主星）：**
- 地星（皇帝的象）：臉寬方正、耳朵不外露、不苟言笑、厚重
- 孤星性質：無左輔右弼在旁 → 孤單皇帝（如宗教人士的象）
- 解惡自化功能：遇到煞星可自解
- 「未言先笑」者 = 小人（反面教材）

**天機星（北斗陰星，第二主星）：**
- 文官帶、陰星
- 外貌：皮膚黑、個子矮小精幹
- 反應快、隨機而動、智慧高
- 無解惡自化功能

**太陽星（第三主星）：**
- 接續上堂，本堂點出其象（紅光滿面）
