## 關鍵概念列表

### 一、命盤案例：民國42年（半空折尺中年凶災）

#### 命盤特徵
- 民國42年男命
- 命盤有：半空折尺 + 火星 + 千宜（畫記）
- 畫記 = 半空折尺（非常凶）
- 積月同梁也有凶災；意外死

#### 半空折尺的時間鎖定
- 半空折尺的凶災線必在中年
- 中年 = 命盤中間那一段大限
- 凶災不用看細節，先把「線」鎖定在中年

#### 與地理配合
- 西北角廚房 → 克夫；西南角廚房 → 克妻
- 命 + 運 + 地理三者疊加才最凶

#### 案例結果
- 民國82年，中南部某政治人物被槍擊致命
- 此人41歲，剛好在中年大限
- 命盤流年：太陽現地（落陷），日月看不到天上的光
- 太陽現地 = 眼睛看不到日月 = 昏迷或死亡
- 倪師評語：「這命解解不掉，差一個禮拜我還沒開始動手，打來說不用批了」

### 二、廚房最佳位置

#### 凶位（不宜放廚房）
- **西北角廚房** → 克夫（史夫）
- **西南角廚房** → 克妻（史妻）

#### 吉位（推薦放廚房）
- **正北方廚房** = 最佳位置
- **正西方廚房** = 次佳位置

### 三、命盤案例：民國47年（日月反背）

#### 日月反背格局
- 父母宮出現日月反背之象（太陰化泉 + 太陽現地）
- 傳統算命師判定：「惡性宜生」= 要過繼給別人才能活
- 現代說法：應離開親生父母

#### 命大者另有解法
- 此人雖有日月反背，卻命大仍活
- 出生那一天，醫院發生火災 → 混亂中被抱錯嬰兒
- 結果：他跟親生父母離散 = 以另一種方式應驗了「惡性宜生」

#### 相學驗證
- 正常人：日角（左額）月角（右額）應稍微凸出
- 此人：日角月角一平（既不凸也不凹）= 命格有異
- 倪師問法：先問「你有沒有跟父母住在一起」，確認後才點破，避免傷害

### 四、命解（消極）vs 運解（積極）

#### 命解的局限
- 命解 = 消極方式，例如：「你婚姻不好，晚點結婚吧」
- 話好聽但做不到，等於廢話
- 大多數算命師只能給命解

#### 運解的價值
- 運解 = 積極改變，通過陽宅位置調整
- 例：男孩子未婚前讓他睡東宮（長子位）
- 腳朝延年方（對應命格的延年方位）
- 延年方有解惡自化的功能 = 解災星

#### 如何選延年方
- 東南西北的延年方因命格而異
- 若房子是正向 = 朝正方向
- 若房子是斜向 = 對著四個角之一（看哪角比較好）
- 不能教他朝太陽落陷的方向 = 可能損傷眼睛

### 五、民國47年命盤細部分析

#### 基本格局
- 天悅星落命 + 對面天奎：做跪相（必有科獎）
- 化權在財伯公：必在私人企業掌權做主管
- 靠專業知識（有科獎技術）在私企做領導

#### 各大限走勢
- **16-25歲（西薩天馬）**：天馬動 = 離開父母外地求學，16歲後高中、大學在外地
- **26-35歲（福德宮）**：26歲逢紅藍星入夫妻宮 = 26歲結婚；巨門星在夫妻 = 娶個個子小但嘮叨的太太；紅藍在夫妻宮=認了，天生愛她
- **36-45歲（田宅宮）**：武曲化科在田宅 + 對面科路 = 房地產慢慢增加；老闆分股利給他
- **46-55歲（日月反倍）**：事業不順，披星戴月，事與願違

#### 中庭相與大限配合
- 看中年運好不好：看「中庭」（面部中間段）是否飽滿
- 上中下三庭相等 = 各階段平順
- 可用流年逐年推算，加減地理位置因素

### 六、辦公室的易經風水

#### 基本原則
- **外掛**：辦公室門的朝向（代表最外卦）
- **內掛**：老闆主位（代表內卦）
- 辦公室門 + 老闆桌 = 組成一個易經卦象

#### 實際應用
- 辦公室門朝西北 = 外卦是乾（天，前）
- 老闆在西南 = 內卦是坤（地）= 天地皮卦（皮子，凶）
- 辦公室門朝西南 + 老闆在西北 = 地天太（通暢，吉）
- 老闆桌位再根據八字的方向調整

### 七、皮卦詳解（天地否卦）

#### 皮卦的卦象
- 外卦：天（乾，往上走）；內卦：地（坤，往下走）
- 天在上繼續往上，地在下繼續往下 = **兩相隔絕，不通**

#### 皮卦的哲學
- 易經本文：「匹茲匪仁」= 不是仁者之道，不是正道
- 三才（天、地、人）中：人 + 地理 = 三分之二 > 天命三分之一
- 意即：人間道和地理各佔三分之一，加起來大於純命格

#### 從太卦到皮卦：物極必反
- 太卦到達終極（上六）= 陰一直在揉 = 反轉成皮卦
- 「物極必反」= 太平盛世終究走向崩壞
- 君王若一昧柔順（只有揉，無威）= 最後小人得道

#### 一念之間
- 太卦（通）和皮卦（不通）相差一念之間
- 外柔內盡 = 太；外似君子內壞水 = 皮

### 八、除皮之道（君子的應對策略）

#### 遇到皮賽不通時
- 皮卦 = 小人當道，君子道消
- 君子的應對：**以退為進，不居榮位**
- 履卦（天則履）可解皮卦：「履的序卦是地天太」
- 處皮之時 = 有履就可解（對人謙和有禮）

#### 在政壇的應用
- 遇到皮賽不通 = 能退出來的都是邪人（閒才）
- 繼續留下拼搏的都是小人
- 「在疲惑的時候，所有小人都是升官的」

#### 小人的特徵
- 外面穿西裝筆挺、說的都是君子話
- 裡面一肚子壞水 = 言行不一致，表裡不一致
- 「無視而笑」= 看到你就笑但沒有說話 = 心生毒氣的壞蛋
- 君子不開口，直接去做，不需要歌功頌德

### 九、皮卦占卜圖像解讀

#### 圖像一：男子臥病（羊病）
- 圖中有男子臥床 = 一定是「裝病」（同音：羊裝病）
- 含義：此人一事無成，用裝病來面對親朋好友
- 或者：政治人物退出政壇的策略是「裝病退休」= 閒才之道

#### 圖像二：進破（鏡子打破）
- 鏡子打破落地 = 破鏡難圓
- **夫妻難和**，夫妻再難團圓
- 若是招（徵招入朝）：不可復（不要去接受），否則被小人利用完再丟棄

#### 圖像三：小人當道（人坐路中）
- 有人坐在路中間，不讓人通過
- 小人群聚（一群，不是一個）= 在皮賽之時最怕被揭發
- 小人會「主在路中心」= 張弓箭頭在地 = 你已被利用完了，趕快退

#### 圖像四：拍掌笑（第三者造謠）
- 一個人在旁邊幸災樂禍地拍掌笑
- 補夫妻時若補到此象 = 夫妻分離有第三者破壞
- 此人特徵：造假造謠，攻擊兩人之間的感情
- 識別方法：鄰居中哪個人看到你總是不說話就在笑？= 就是他

### 故事1：被抱錯嬰兒 — 日月反背的另類應驗
- **類型**：命理神準 / 震撼案例
- **核心**：民國47年命盤有日月反背，傳統算法要「過繼」才能活。此人命大沒過繼，卻在出生那天因醫院火災，被抱錯嬰兒，意外與親生父母分離——命格以另一種方式應驗
- **震撼點**：倪師不直接問「你跟父母住一起嗎？」，先觀察日角月角的面相確認後才點破
- **金句**：「這命解解不掉，差一個禮拜，他打來說不用批了。」
- **二創建議**：「命格裡的事情一定會發生，只是方式不同」——從日月反背案例解析命格應驗的多種形式

### 故事2：命解消極，運解積極 — 陽宅調整才是真正的改運
- **類型**：命理方法論 / 實用智慧
- **核心翻轉**：大多數算命師只能給「命解」（晚點結婚、避免某事），這是消極的。倪師用「運解」（陽宅延年方位調整）積極化解凶星
- **反常識**：改善命運不是靠符咒，而是靠睡覺的方向和房間位置
- **金句**：「命解是消極的，運解才是積極的。」
- **二創建議**：「算命師說你命不好，這不是答案」——命解與運解的根本差異

### 故事3：無視而笑 — 識破小人的終極方法
- **類型**：哲學智慧 / 人際識鑑
- **核心**：皮卦時代小人當道，識別方法極簡單：「見到你不說話，就在那裡笑的人 = 心生毒氣的壞蛋」
- **金句**：「無視而笑 = 心生毒氣，你沒事笑什麼？壞蛋。」
- **二創建議**：「如何辨認身邊的小人？」——從皮卦占卜圖解析識人術的五種方法

### 故事4：廚房位置決定誰先走
- **類型**：陽宅案例 / 震撼點
