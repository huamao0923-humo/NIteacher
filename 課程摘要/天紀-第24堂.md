## 關鍵概念列表

### 一、名位相等（深入）

#### 核心原理
- 你的「名」（身份、名義）= 你的「位」（居住或辦公位置）
- 名等於位 = 加和（加分）；名不等於位 = 扣分
- **你坐誰的位子，要用誰的名字**

#### 公司掛名應用
- 先生的名字 + 先生住西北角 = 事業運旺
- 先生的名字 + 先生住二兒子位（東南）= 天水宋卦 = 準備打官司
- 太太的名字 + 先生住太太位（西南）= 如地火明夷 = 損女
- 解決方案：先生去住西北，公司改掛先生的名

#### 看大宮發哪一房
- 一進門看哪個宮（房間）最大，就代表那一房（角色）最強、最發
- 例：東宮最大 = 長子最強，要先巴結長子；西南最大 = 婆婆說了算

### 二、四代交替（世代傳承）

#### 正確的傳承順序
1. **目前：父母住西北角**（名=夫妻，位=西北，相符）
2. **傳給已婚兒子**：
   - 兒子（含媳婦）搬到西北角（名=夫妻，位=西北）
   - 父母退位到三兒子位（天山掛），享清閒
3. **傳給未婚兒子**：
   - 兒子先住中宮（名位相等暫時處理）
   - 等兒子結婚後再搬到西北角

#### 關鍵原則
- 繼承位要名位相等才能「師出有名」
- 若名不對位，事業必然打折甚至出問題

#### 南北半球差異
- 北半球：西北=夫妻，東=長子，西南=母
- 南半球：南北方向對調，中間東西不動
- 先學好北半球，南半球一詞類推

### 三、廚房位置凶吉

- **西北角廚房** → 死先生（不吉）
- **西南角廚房** → 死妻（不吉）
- **西偽方客廳**（客廳在偏西方）→ 夫妻生離

> 相關命格需搭配「命上有哉」和「運上有哉」才能確認，三分之二才大於人事力量。

---

### 四、夫妻死別的判斷（相學+命理）

#### 眼相判斷
- **魚尾紋往下走** → 生離之相
- **魚尾紋處有渣點或暗黑** → 死別之相
- **白色紋路在眼尾** → 對方已死（或將死）

#### 面相性別差異
- **男命**：哭臉之人 = 夫妻死別或生離之相
- **女命**：面大骨粗 = 夫妻死別之相

#### 臨終前的相
- 死前兩週內面色會變：紅色基底但毛孔泛青（稱「沙奪煙罩」）= 死神近身
- 倪師教主委：事前要觀察，可以事先計算最危險的日期
- 可用液晶掛（卦）算出死亡日期，提前做準備

#### 生離vs死別的命格差異
- 命格上連破/連癱/地水師等星在福德宮 = 夫妻死別或生離
- 命局三分之一，地理三分之一，人間道三分之一
- 三者加起來大於二才真正無法改變

## 🎬 二創素材

### 故事1：名位相等 — 公司掛誰的名，誰就住誰的位
- **類型**：陽宅學 / 實用命理
- **核心翻轉**：不是「先生辛苦創業就用先生的名字」，而是「誰住西北角就用誰的名字」
- **具體案例**：先生名字但住在二兒子位（東南）= 天水訟卦 = 準備打官司；移到西北角 = 事業旺
- **金句**：「你坐誰的位子要用誰的名字，位等於名才能發事業。」
- **二創建議**：「為什麼你的公司取了好名字還是不發？」——從名位相等解析陽宅與事業的隱形關係

### 故事2：四代交替 — 傳承的陽宅學秘密
- **類型**：陽宅學 / 家族傳承
- **核心**：傳家業不只是法律繼承，還要「名位相等」才能師出有名。兒子繼承前需先搬到西北角，父母退位到三兒子位（享清閒）
- **反常識**：父母不肯退到三兒子位，兒子就算繼承也名不正言不順，事業打折
- **二創建議**：「為什麼富二代接班常常失敗？」——從陽宅四代交替解析傳承的隱形規律

### 故事3：魚尾紋洩露夫妻命運
- **類型**：相學神準 / 夫妻緣分
- **核心**：
  - 魚尾紋往下走 = 生離（分離但不死）
  - 魚尾紋有渣點或暗黑 = 死別
  - 白色紋路在眼尾 = 對方已死或將死
- **震撼點**：只看眼角，就能判斷是離婚還是喪偶
- **二創建議**：「你的魚尾紋在說什麼？」——相學解析眼角紋路與夫妻緣分的對應
